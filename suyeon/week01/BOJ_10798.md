### 🧩 풀이 설계 / 막힌 부분
- 처음 접근 아이디어
> 2차원 배열 선언하고 넣음  
index 0 ~ 가장 큰 index까지 각 array를 순회하면서 문자열 만들기  
중간에 없는 문자라면? -> 해당 줄의 길이보다 작은 index일때만 sb에 넣으면 됨

- 고려한 시간복잡도: O(N)

- TIL: Java에서 char 타입의 배열을 생성하면 모든 요소는 기본값인 `\0` 으로 자동 초기화된다.

### ⭐️ 문제에서 주로 사용한 알고리즘
- 구현, 문자열, 2차원 배열

### 🧠 대략적인 코드 설명
```java
public class Main {

    public static void main(String[] args) throws Exception {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        StringBuilder sb = new StringBuilder();

        char[][] arr = new char[5][];
        int maxLength = 0;

        for (int i = 0; i < 5; i++) {
            arr[i] = br.readLine().toCharArray();
            int length = arr[i].length;
            if (length > maxLength){
                maxLength = length;
            }
        }
        // System.out.println(Arrays.deepToString(arr));
        // 넣기 완료

        for (int i = 0; i < maxLength; i++){
            for (int j = 0; j < 5; j++){
                if (i < arr[j].length) {
                    sb.append(arr[j][i]);
                }
            }
        }

        System.out.println(sb);

    }
}
```
