### 🧩 풀이 설계 / 막힌 부분

#### 문제 분석

- 전체 학생의 수는 2명 이상 30명 이하입니다.
- 체육복을 도난당한 학생의 수, 여벌의 체육복을 가져온 학생의 수는 1명 이상 n명 이하이고 중복되는 번호는 없습니다.
- 최대한 많은 학생이 수업을 듣는 게 목표입니다.
- 체육복을 도난 당한 학생의 앞뒤 번호의 여벌 학생이 해결 가능함

#### 설계

- count = n으로 변수 생성
- lost(i), reserve(j) 배열 정렬한 후 2중 for 문
- i를 해결할 수 있는 j가 없다면 count 감소
- 최종 count 값 도출

-> 다시 set으로 수정함
-> 존재 여부를 탐색

### ⭐️ 문제에서 주로 사용한 알고리즘

Set

### 🧠 대략적인 코드 설명

```js
function solution(n, lost, reserve) {
    const lostSet = new Set(lost);
    const reserveSet = new Set(reserve);

    // 중복 삭제
    for (let e of lostSet){
        if(reserveSet.has(e)){
            lostSet.delete(e);
            reserveSet.delete(e);
        }
    }
    
    for (let e of reserveSet) {
        if (lostSet.has(e-1)){
            lostSet.delete(e-1)
        } else if (lostSet.has(e+1)){
            lostSet.delete(e+1);
        }
    }
    
    return n - lostSet.size;
}
```
