# 💯 Algorithm & Coding Test Study

> 각자 다른 문제를 풀고, 자동화된 구조로 기록을 남기는 개인 중심 알고리즘 스터디

* 플랫폼 무관 (BOJ / Programmers / LeetCode 등)
* PR 없음, 코드 리뷰 강제 없음
* **자동 폴더 생성 + 문제 풀이 기록 템플릿 제공**
* “꾸준히 풀고, 정리하고, 남긴다”에만 집중

---

## 🎯 스터디 목표

* 알고리즘/코딩 테스트 문제를 **지속적으로 풀이**
* 풀이 과정을 **구조화된 형태로 기록**
* 주차 단위로 정리하여 **회고 가능한 로그 축적**
* Git을 이용한 **개인 학습 이력 관리**

---

## 🧭 전체 흐름 요약

1. 저장소를 `clone`
2. 최초 1회, **본인 이름(or GitHub ID) 폴더 생성**
3. 매주 명령어 한 번으로:

   * 해당 주차 폴더 생성
   * 문제 풀이 파일 + 템플릿 자동 생성
4. 문제 풀이 후 **정해진 커밋 규칙으로 커밋**

---

## 📁 Repository 구조

```text
{이름 또는 GitHubID}/
 ├─ week01/
 │   ├─ BOJ_2557.md
 │   └─ PGS_문제이름.md
 ├─ week02/
 │   └─ ...
 └─ ...
```

* **개인 폴더는 최초 1회만 생성**
* 이후 모든 작업은 **본인 폴더 내부에서만**
* 주차 기준으로 정리 (`week01`, `week02`, …)

---

## ⚙️ 자동화: 주차 폴더 & 파일 생성

### 사용 목적

* 매주 반복되는 폴더/파일 생성 작업 제거
* 정해진 템플릿으로 **기록 품질 통일**

### 사용 방법

```bash
create {이름} {주차} {플랫폼} {문제번호}
```

#### 예시

```bash
create chaewon week03 BOJ 2557
```

#### 실행 결과

```text
chaewon/
 └─ week03/
    └─ BOJ_2557.md
```

> alias가 동작하지 않는 경우

```bash
./scripts/create.sh {이름} {주차} {플랫폼} {문제번호}
```

## 📝 문제 풀이 파일 템플릿

자동 생성되는 파일에는 아래 템플릿이 포함된다.

```md
### 📖 풀이한 문제
- 백준 2557 - Hello World
- 난이도: 브론즈 5
- 걸린 시간: 1분 (optional)

### ⭐️ 문제에서 주로 사용한 알고리즘
- 출력 / 구현

### 🧠 대략적인 코드 설명
- 표준 출력으로 문자열을 출력하는 문제
- 입출력 형식 확인 후 바로 구현
```

> 항목 추가/ 삭제 자유

## 🧾 커밋 규칙 (중요)

### 커밋 메시지 형식

```
[플랫폼] 문제이름 / 난이도 / 걸린시간
```

#### 예시

```bash
git commit -m "[BOJ] Hello World / 브론즈5 / 1분" \
            -m "https://www.acmicpc.net/problem/2557"
```

### 플랫폼 태그 통일

| 태그      | 플랫폼                  |
| ------- | -------------------- |
| `[BOJ]` | 백준                   |
| `[PGS]` | 프로그래머스               |
| `[LTC]` | 리트코드                 |
| `[CFS]` | 코드포스                 |
| `[SEA]` | 삼성 SW Expert Academy |
| `[ETC]` | 기타                   |
