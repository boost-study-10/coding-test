### 📖 풀이한 문제

- 백준 2493 - 탑
- 문제 링크: https://www.acmicpc.net/problem/2493
- 난이도: gold5
- 걸린 시간: 10m

### 🧩 풀이 설계 / 막힌 부분

- 처음 접근 아이디어: 이중 for 문
- 고려한 시간복잡도: O(n^2) -> O(n)
- 막힌 부분 / 해결 방법:
  - 이중 반복문으로 풀면 구현 문제처럼 쉽게 풀 수 있긴 하지만 n이 굉장히 커서 시간초과가 난다.
  - stack을 활용해 풀었다.
  - 탑을 앞에서부터 확인하고, stack이 비어있으면(이 안에 든 게 비교군) 바로 push
  - stack에 비교군이 있으면 뒤에서부터 지금 탑과 높이를 비교한다. 왼쪽으로 레이저를 쏘기 때문에 뒤쪽에 있는 것 부터 비교해야함. 만약 본인보다 작으면 어차피 레이저를 못받기 때문에 pop, 받을 수 있다면(높이가 자신보다 높다면) 해당 탑의 인덱스를 답으로 저장

### ⭐️ 문제에서 주로 사용한 알고리즘

- stack

### 🧠 대략적인 코드 설명

- python

```
import sys
input = sys.stdin.readline

n = int(input())
heights = list(map(int, input().split(" ")))

stack = []
answer = [0] * n
for i in range(n):
    while stack:
        if stack[-1][1] > heights[i]:
            answer[i] = stack[-1][0] + 1
            break
        else:
            stack.pop()
    stack.append((i, heights[i]))

print(*answer)
```
