### 📖 풀이한 문제

- 백준 1182 - 부분 수열의 합
- 문제 링크: https://www.acmicpc.net/problem/1182
- 난이도: silver 2
- 걸린 시간: 15m

### 🧩 풀이 설계 / 막힌 부분

- 처음 접근 아이디어: 재귀 탐색, 완전 탐색
- 고려한 시간복잡도: O(2^n)
- 막힌 부분 / 해결 방법:

### ⭐️ 문제에서 주로 사용한 알고리즘

- dfs, 완전탐색

### 🧠 대략적인 코드 설명

- Python

```
import sys
input = sys.stdin.readline
from collections import deque


def dfs(idx, sum):
    global answer

    if idx == n:
        if sum == s:
            answer += 1
        return

    dfs(idx+1, sum)
    dfs(idx+1, sum+nums[idx])


n, s = map(int, input().split())
nums = list(map(int, input().split()))
answer = 0
dfs(0,0)

if s == 0:
    answer -= 1

print(answer)
```
