- 백준 2098 - 외판원 순회
- 문제 링크: https://www.acmicpc.net/problem/2098
- 난이도: gold1
- 걸린 시간: 50m

### 🧩 풀이 설계 / 막힌 부분

- 처음 접근 아이디어: 비트마스킹으로 방문표시 + dfs로 완탐
- 고려한 시간복잡도:
- 막힌 부분 / 해결 방법:
  - 처음에 dfs로 거의 완탐을 했다. 경우의 수가 n! 수준이라 n이 최대 16, 그럼 16! = 2e13 수준이라 풀 수 없었음 시간초과 났다.
  - 알고리즘 분류를 보고 힌트를 얻었다. dp + 비트마스킹 문제엿슨!
  - `dp[현재도시][방문상태]` 로 두고, 남은 도시를(방문상태에 표시 안된 것들) 모두 방문하는 최소 비용을 `dp[now][visit]`을 메모이제이션 했다.
  - 그럼 도시가 5개라고 치면, 0->1->2->3 상태랑 0->2->1->3 일때는 dp가 같다. 왜냐면 현재도시가 3이고, 방문상태에 0123을 방문한 것으로 되어있기 때문. 두 경우 모두 지금까지 지나온 경로비용 + `dp[3][01111]`를 하면 최소비용을 구할 수 있다.

### ⭐️ 문제에서 주로 사용한 알고리즘

- dp, 비트마스킹

### 🧠 대략적인 코드 설명

- python

```
import sys
input = sys.stdin.readline

INF = 10**9

def dfs(now, visit):
    # 모든 도시 방문
    if visit == (1<<n) - 1:
        if weights[now][0] == 0:
            return INF
        return weights[now][0]

    # 이미 계산된 상태
    if dp[now][visit] != -1:
        return dp[now][visit]

    dp[now][visit] = INF

    for next in range(n):
        # 이미 방문
        if visit & (1<<next):
            continue
        # 길 없음
        if weights[now][next] == 0:
            continue

        cost = weights[now][next] + dfs(next, visit|(1<<next))
        dp[now][visit] = min(dp[now][visit], cost)

    return dp[now][visit]

n = int(input())
weights = [list(map(int, input().split())) for _ in range(n)]
dp = [[-1]*(1<<n) for _ in range(n)]

print(dfs(0, 1))
```
