### 📖 풀이한 문제

- 백준 1912 - 연속합
- 문제 링크: https://www.acmicpc.net/problem/1912
- 난이도: silver2
- 걸린 시간: 10m

### 🧩 풀이 설계 / 막힌 부분

- 처음 접근 아이디어: 음수 빼고 양수끼리 가장 큰 수열 구하기
- 고려한 시간복잡도: O(n)
- 막힌 부분 / 해결 방법:
  - 양수+음수를 해도 양수 하나보다는 작은 값이 나오니까 음수를 빼고 양수중에 가장 큰 합을 만드는 수열을 구할 거라고 생각했다.
  - 하지만 [-2, 3, -1, 5] 이런 수열에서 음수를 버리면 가장 큰 값은 5, 음수 포함하면 7이 나올 수 있다.
  - dp[i]는 i에서 끝나는 연속합의 최대값이라고 생각하고 점화식을 짰다. 새로 시작할지, 혹은 이어서 갈지를 선택할 수 있도록 점화식은 다음과 같다.
    `dp[i] = max(nums[i], dp[i-1] + nums[i])`

### ⭐️ 문제에서 주로 사용한 알고리즘

- dp

### 🧠 대략적인 코드 설명

- python

```
import sys
input = sys.stdin.readline

n = int(input())
nums = list(map(int, input().split(" ")))

dp = nums[:]

for i in range(1, n):
    dp[i] = max(nums[i], dp[i-1] + nums[i])

print(max(dp))
```
