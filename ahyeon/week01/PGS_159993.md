### 🧩 풀이 설계 / 막힌 부분

- 간단 문제 설명:
  - start에서 시작해 레버를 거쳐 exit로 도착하는 최단거리 반환
  - o칸은 갈 수 있고 x인 곳은 지나갈 수 없음
  - exit에 도착할 수 없다면 -1 반환
- 처음 접근 아이디어:
  - 간단 bfs 문제
  - 레버 flag를 하나 두고 레버를 지나가면 true로 바꾸기
- 고려한 시간복잡도:
- 막힌 부분 / 해결 방법:
  **막힌 부분**
  - start 좌표만 queue에 넣어두고 레버를 지나가기 전까지는 flag를 false로 두고, flag==false이면 exit를 도착, 경유 하더라도 무시하고 flag=true일때만 exit에 도착한 걸로 치려고 했다.
  - 위 방식대로 하면 상태가 섞인 그래프 탐색이 된다. 풀이가 가능은 한데 방문처리, 거리처리, 상태분기 등등을 해야해서 풀이가 꼬였다...

  **해결 방법**
  - 상태를 섞지 않고 구간을 나눠서 풀기로 했다. 정석적인 구조도 이에 가깝지 않을까..
  - 무조건 S에서 시작 L을 거쳐 E로 가야한다. 따라서 (S->L) 경로의 bfs 한 번, (L->E) 경로의 bfs 한 번을 따로 구해 거리를 합하는 방식으로 풀이했다.

- 파이썬 vs 자바스크립트:
  **파이썬**
  - 파이썬은 deque라는 좋은 queue 라이브러리를 제공한다.
  - queue로 bfs를 구현하는 정석적인 방법으로 풀었다.

  **자바스크립트**
  - 자바스크립트에서 queue를 구현해야할까? 아니 난 못한다.
  - 자바스크립트의 배열에서 `shift`라는 메서드를 사용하면 배열의 맨 앞 요소를 빼내는 파이썬의 deque.popleft() 와 같이 작동하긴한다.
  - 하지만 배열의 맨 앞을 빼면서 뒤에 있는 모든 원소를 한 칸씩 당기기 때문에 시간복잡도가 `O(n)`이 되어 시간이 터질 수 있다.. 파이썬의 dp.popleft()는 `O(1)`
  - 따라서 배열에 push만 하고 인덱스를 증가시키는 방법을 사용한다. `queue[head++]` 이런식으로 head라는 포인터를 만들어 사용하는 것. (사실 deque의 popleft는 이것처럼 포인터 이동과 같은 방식이다 그래서 O(1))

### 🧠 대략적인 코드 설명

- python

  ```
  from collections import deque

  def bfs(start, end, maps, n, m):
      visit = [[-1] * m for _ in range(n)]
      q = deque()
      q.append(start)
      visit[start[0]][start[1]] = 0
      dirs = [[0,1], [0,-1], [1,0], [-1,0]]
      while len(q) > 0:
          ci, cj = q.popleft()
          cv = visit[ci][cj]
          for di, dj in dirs:
              ni, nj = ci+di, cj+dj
              if (0<=ni<n and 0<=nj<m):
                  if (maps[ni][nj] == "X"): continue
                  if (ni==end[0] and nj==end[1]):
                      return cv+1
                  else:
                      if visit[ni][nj] == -1:
                          visit[ni][nj] = cv + 1
                          q.append([ni, nj])

  def solution(maps):
      n, m = len(maps), len(maps[0])
      start, end, lever = [], [], []

      # find start point
      for i in range(n):
          if (start and end and lever): break
          for j in range(m):
              if maps[i][j] == 'S':
                  start = [i, j]
              elif maps[i][j] == 'E':
                  end = [i, j]
              elif maps[i][j] == 'L':
                  lever = [i, j]

      stol = bfs(start, lever, maps, n, m)
      ltoe = bfs(lever, end, maps, n, m)

      if not stol or not ltoe:
          return -1
      else:
          return stol + ltoe
  ```

- javascript

  ```
  function bfs(start, end, maps, n, m) {
      const q = []
      q.push(start)
      let head = 0
      const visit = Array.from({ length: n }, () => Array(m).fill(-1));
      visit[start[0]][start[1]] = 0
      const dirs = [[0,1], [1,0], [0,-1], [-1,0]]
      while (q.length > head) {
          const [ci, cj] = q[head]
          const cv = visit[ci][cj]
          for (const [di, dj] of dirs) {
              const ni = ci+di, nj = cj+dj
              if (0<=ni && ni<n && 0<=nj && nj<m) {
                  if (ni === end[0] && nj === end[1]) {
                      return cv + 1
                  } else if (maps[ni][nj] === "X") {
                      continue
                  } else if (visit[ni][nj] === -1) {
                      visit[ni][nj] = cv + 1
                      q.push([ni, nj])
                  }
              }
          }
          head++
      }

  }

  function solution(maps) {
      const n = maps.length
      const m = maps[0].length
      let start = [], end = [], lever = []
      for (let i=0; i<n; i++) {
          for (let j=0; j<m; j++) {
              if (maps[i][j] ==='S') {
                  start = [i, j]
              } else if (maps[i][j] === 'L') {
                  lever = [i, j]
              } else if (maps[i][j] === 'E') {
                  end = [i, j]
              }
              if (start.length>0 && end.length>0 && lever.length>0) break
          }
          if (start.length>0 && end.length>0 && lever.length>0) break
      }
      const stol = bfs(start, lever, maps, n, m)
      const ltoe = bfs(lever, end, maps, n, m)

      if (!stol || !ltoe) {
          return -1
      } else {
          return stol + ltoe
      }

  }
  ```
