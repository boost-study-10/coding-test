- 문제 링크: https://school.programmers.co.kr/learn/courses/30/lessons/389481

### 🧩 풀이 설계 / 막힌 부분

- 간단한 문제 설명:
  - 전설 속의 비밀 주문서에는 `'a' -> 'b' -> ... -> 'z' -> 'aa' -> 'ab' -> ... -> 'az' -> 'ba' -> ... -> 'zz' -> 'aaa' -> ...` 순서로 정렬된 알파벳 소문자 11글자 이하의 각 주문이 있다. 몇개의 주문을 삭제 했을 때 n번째 주문이 무엇인지 찾아야 한다.
- 처음 접근 아이디어:
  - 각 자릿수에 a~z까지 26개의 문자가 올 수 있다.
  - 11자리까지 있다고 했으니 1자리면 26개의 경우의 수, 2자리면 26^2, .. 각 자릿수의 경우의 수를 구한다. `lenarray = [26, 676, 17576, ...]`
  - 삭제할 주문이 abc라면 `lenarray[3-1]*1 + lenarray[3-2]*2 + lenarray[3-3]*3` 처럼 `lenarray[주문자릿수-인덱스]*(알파벳순서)` 를 전부 더하면 해당 주문의 순서를 구할 수 있다.
  - 삭제할 주문이 n보다 작다면 순서에 영향을 끼치고, 그렇게 때문에 n을 하나씩 키운다.
  - 순서에 영향을 주는 주문들의 수만큼 n에 구한 후 0이 없는 26진법으로 주문을 구한다.
- 고려한 시간복잡도:
- 막힌 부분 / 해결 방법:

  **막힌 부분**
  - 0이 없는 진법이 아니라 0~25을 한자리로 쓰는 26진법으로 구해 오류가 났다.
  - a=1, b=2, …, z=26 처럼 0이 없는 진법으로 구해야한다.
  - 처음엔 `n // 26, n % 26` 이런 식으로 풀이를 구성했는데, 만약 순서라 52라면 az가 나와야 하지만 저 풀이대로 하면 b\_가 나온다 (순서를 \_abcdefghijklmnopqrstuvwxyz 이렇게 잡아놓음)

  **해결 방법**
  - 간단하게 n-1 을 하고 같은 식을 써주면 된다.
  - 이런 스타일의 문제는 무조건 아래의 공식을 떠올리기
    ```
    n -= 1
    digit = n % 26
    n //= 26
    ```

### ⭐️ 문제에서 주로 사용한 알고리즘

- 수학 + 구현
- 진법 변형, 0이 없는 진법

### 🧠 대략적인 코드 설명

- `checkleftn` : bans의 요소들 중 n번째 전에 오는(순서에 영향을 끼치는) 주문인지 확인
- `makechar` : 구해야하는 순서의 주문 만들기 (0이 없는 26진법으로 구현)

  ```
  def checkleftn(char, lenarray, dic, n):
      seq = 0
      L = len(char)
      for i in range(L):
          seq += lenarray[L-i-1]*dic[char[i]]
      return seq <= n

  def makechar(n, lenarray):
      result = ''
      arr = 'abcdefghijklmnopqrstuvwxyz'

      while n > 0:
          n -= 1
          result = arr[n%26] + result
          n //= 26

      return result

  def solution(n, bans):
      answer = ''
      bans.sort(key=lambda x: (len(x), x))
      lenarray = [26**i for i in range(12)]
      dic = {"a":1, "b":2, "c":3, "d":4, "e":5, "f":6, "g":7, "h":8, "i":9, "j":10, "k":11, "l":12, "m":13, "n":14, "o":15, "p":16, "q":17, "r":18, "s":19, 't':20, 'u':21, "v":22, "w":23, "x":24, "y":25, "z":26}

      for ban in bans:
          if checkleftn(ban, lenarray, dic, n):
              n += 1

      answer = makechar(n, lenarray)
      return answer
  ```
