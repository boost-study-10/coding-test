### 🧩 풀이 설계 / 막힌 부분

- 처음 접근 아이디어: 2차원 bfs
- 고려한 시간복잡도:
- 막힌 부분 / 해결 방법:

  **막힌 부분**

  단순 2차원 bfs로 생각하고 풀이했다.(`visit[i][j]`) 해당 문제는 칸에 접근하는 방향에 따라서 비용이 달리지는 문제라 접근하는 방향을 함께 저장하고 비교해야했는데, 이를 생각하지 않고 단순 비용만 생각해 몇 케이스에서 실패했다.

  ***

  **해결 방법**

  같은 칸에 도착했더라도 위에서, 왼쪽에서, 오른쪽에서, 아래에서 들어온 경우에 따라 비용이 달라지기 때문에 각 방향에 따른 값도 서로 다른 상태로 유지했다.
  `visit[i][j] => visit[i][j][dir]`
  마지막 칸에 도착하기까지의 최소 비용을 구하는 문제기 때문에, 마지막 칸(`visit[n-1][n-1]`) 중에서 최소값을 구해 완성했다.

  ***

  **정석 풀이**

  나는 bfs를 생각하다 발전시킨 풀이로 `큐(deque) + 갱신되면 다시 큐에 넣기` 방식인 SPFA(Shortest Path Faster Algorithm)로 풀이했다. 즉 들어온 순서대로 처리하는 풀이방식이다.
  시간복잡도는 `O(V*E)` 로 문제를 대입하면 `O(n^2 * n^2) = O(n*4)`. 평균은 빠를 수 있지만 최악이 진짜 나빠짐.. 해당 문제에서는 n이 아주 크지 않고 상태도 4배(동서남북) 뿐이라서 SPFA방식도 통과되었다.

  정석의 방식은 다익스트라 알고리즘을 이용하는 것이다.
  다익스트라 알고리즘은 queue대신 heap(우선순위큐)을 이용한다. 따라서 비용이 가장 싼 것 부터 처리하기 때문에 확정(이 상태는 이제 끝. 더 싸게 오는 일은 없음)이라는 개념이 생겨 시간복잡도를 줄일 수 있다.
  다익스트라로 풀이하면 시간복잡도는 `O(E log V)`. 이 문제로 대입하면 `O((n^2) log(n^2))` 정도

### ⭐️ 문제에서 주로 사용한 알고리즘

- bfs, spfa

### 🧠 대략적인 코드 설명

- spfa

```
from collections import deque

def bfs(n, board):
    q = deque()

    # visit[i][j][d]
    visit = [[[-1]*4 for _ in range(n)] for _ in range(n)]

    dirs = [[0,1], [1,0], [-1,0], [0,-1]]

    if board[0][1] == 0:
        visit[0][1][0] = 100
        q.append([0,1,0])

    if board[1][0] == 0:
        visit[1][0][1] = 100
        q.append([1,0,1])

    while q:
        ci, cj, cd = q.popleft()
        cv = visit[ci][cj][cd]

        for d in range(4):
            di, dj = dirs[d]
            ni, nj = ci+di, cj+dj

            if (0<=ni<n and 0<=nj<n and board[ni][nj]==0):
                cost = cv + (100 if cd == d else 600)
                if (visit[ni][nj][d] == -1 or visit[ni][nj][d] > cost):
                    visit[ni][nj][d] = cost
                    q.append([ni, nj, d])

    return min(x for x in visit[n-1][n-1] if x != -1)


def solution(board):
    answer = 0
    n = len(board)
    answer = bfs(n, board)

    return answer
```

- 다익스트라

```
import heapq

def bfs(n, board):
    heap = []

    # visit[i][j][d]
    visit = [[[-1]*4 for _ in range(n)] for _ in range(n)]

    dirs = [[0,1], [1,0], [-1,0], [0,-1]]

    # 오른쪽
    if board[0][1] == 0:
        visit[0][1][0] = 100
        heapq.heappush(heap, [100, 0, 1, 0])

    # 아래
    if board[1][0] == 0:
        visit[1][0][1] = 100
        heapq.heappush(heap, [100, 1, 0, 1])

    while heap:
        cost, ci, cj, cd = heapq.heappop(heap)

        if visit[ci][cj][cd] < cost:
            continue

        for d in range(4):
            di, dj = dirs[d]
            ni, nj = ci+di, cj+dj

            if (0<=ni<n and 0<=nj<n and board[ni][nj]==0):
                new_cost = cost + (100 if cd == d else 600)
                if (visit[ni][nj][d] == -1 or visit[ni][nj][d] > new_cost):
                    visit[ni][nj][d] = new_cost
                    heapq.heappush(heap, [new_cost, ni, nj, d])

    return min(x for x in visit[n-1][n-1] if x != -1)


def solution(board):
    answer = 0
    n = len(board)
    answer = bfs(n, board)

    return answer
```
