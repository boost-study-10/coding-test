### 🧩 풀이 설계 / 막힌 부분

- 간단 문제 설명:
  - A와 B가 n/2개씩 주사위를 가져가서 주사위를 굴려서 나온 수를 모두 합해 점수를 계산하고, 점수가 더 큰 쪽이 승리한다. 점수가 같으면 무승부
  - 주사위는 6면이며, 각 면이 나올 확률은 동일, 1~n의 번호를 가지며, 구성은 모두 다르다/
- 처음 접근 아이디어:
  - 조합을 구하고 모든 경우의 수를 계산해서 A가 B를 이기도록 하는 인덱스 조합을 찾자
- 고려한 시간복잡도:
- 막힌 부분 / 해결 방법:
  - 모든 주사위의 면을 조합할 때 중첩 반복문을 써야하나 고민
  - 파이썬에서는 아주 좋은 itertools라는 라이브러리 모음이 있다
  - itertools에는 아래와 같은 아주 좋은 함수들이 있다!!
    - combinations(조합): 순서 없이, 중복없이, r개 고르기
    - product(데카르트 곱): 모든 경우의 수, 중복 허용, 순서 중요
    - permutations(순열): 순서 중요, 중복 없이
    - combinations_with_replacement: 순서없고 중복 허용
  - 이 combinations, product를 이용해 풀이했다.

### ⭐️ 문제에서 주로 사용한 알고리즘

- 조합 + 완전탐색 + 이분탐색

### 🧠 대략적인 코드 설명

```
from itertools import combinations, product
from bisect import bisect_left

def solution(dices):
    answer = []

    dic = {}
    L = len(dices)
    for A_index in combinations(range(L), L//2):
        B_index = [i for i in range(L) if i not in A_index]

        A, B = [], []
        for order_product in product(range(6), repeat=L//2):
            A.append(sum(dices[i][j] for i, j in zip(A_index, order_product)))
            B.append(sum(dices[i][j] for i, j  in zip(B_index, order_product)))

        B.sort()

        wins = sum(bisect_left(B, num) for num in A)
        dic[wins] = list(A_index)

    max_key = max(dic.keys())
    return [x+1 for x in dic[max_key]]
```
