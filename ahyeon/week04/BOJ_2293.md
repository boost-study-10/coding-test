### 🧩 풀이 설계 / 막힌 부분

- 처음 접근 아이디어: dp
- 고려한 시간복잡도: n \* k
- 막힌 부분 / 해결 방법:
  - 무한 배낭
  - 배낭문제 같지만 앞에서 순회하고, 같은 물건을 여러번 사용할 수 있다는 것이 포인트

### ⭐️ 문제에서 주로 사용한 알고리즘

- dp

### 🧠 대략적인 코드 설명

- python

```
import sys
input = sys.stdin.readline

n, k = map(int, input().split())
coins = [int(input()) for _ in range(n)]

dp = [0] * (k+1)
dp[0] = 1

for coin in coins:
    for w in range(coin, k+1):
        dp[w] += dp[w - coin]

print(dp[k])
```
