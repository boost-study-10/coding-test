### 📖 풀이한 문제

- 백준 12865 - 평범한 배낭
- 문제 링크: https://www.acmicpc.net/problem/12865
- 난이도: gold 5
- 걸린 시간: 30m

### 🧩 풀이 설계 / 막힌 부분

- 처음 접근 아이디어: dp
- 고려한 시간복잡도: n\*k
- 막힌 부분 / 해결 방법:
  - 평범한 배낭문제였다. 물건을 넣을지 말지 정해야함.
  - 중첩반복문 중 안쪽에 있는 반복문이 뒤에서부터 도는데, 이는 이전 상태만 참조하기 위함
  - 앞에서부터 돌면 같은 물건이 여러번 사용될 수 있음(물건 1개씩만 있음)

### ⭐️ 문제에서 주로 사용한 알고리즘

- dp(배낭문제)

### 🧠 대략적인 코드 설명

- python

```
n, k = map(int, input().split())
items = [tuple(map(int, input().split())) for _ in range(n)]

dp = [0] * (k+1)

for weight, value in items:
    for w in range(k, weight-1, -1):
        dp[w] = max(dp[w], dp[w-weight] + value)

print(dp[k])
```
