### 📖 풀이한 문제

- 백준 2240 - 자두나무
- 문제 링크: https://www.acmicpc.net/problem/2240
- 난이도: gold4
- 걸린 시간: 30m

### 🧩 풀이 설계 / 막힌 부분

- 처음 접근 아이디어: dp
- 고려한 시간복잡도: O(T*W), 최대 O(1000*30)
- 막힌 부분 / 해결 방법:
  - 1번, 2번 나무를 왔다갔다 하면서 자두를 먹음
  - 1번에서 시작하니까 홀수번 움직이면 2번나무에 가있고, 짝수번 움직이면 1번나무에 있게됨
  - dp[t][w] 의 2차원 배열에 상태 저장 t초에 w번 움직였을 때 먹을 수 있는 최대 자두 개수
  - 이동 안하면 1초 전이랑 움직인 횟수 같으니까 dp[t-1][w] 과 같게 됨
  - 만약 이동하면, w를 하나 늘려야 하니까, 1초 전 w-1과 비교, 이동안한거랑 이동한것중에 더 큰걸로 저장(최대 자두 개수 구하는 중)
  - 그리고 이동을 했든 안했든 거기에 자두가 떨어질 예정이면(홀수 짝수로 구분) + 1

### ⭐️ 문제에서 주로 사용한 알고리즘

- dp

### 🧠 대략적인 코드 설명

- python

```
import sys
input = sys.stdin.readline

T, W = map(int, input().split())
info = list(int(input()) for _ in range(T))

dp = [[0]*(W+1) for _ in range(T+1)]
# 홀수번 움직이면 2번, 짝수번 움직이면 1번

for t in range(1, T+1):
    for w in range(W+1):
        # 1. 이동 안함
        dp[t][w] = dp[t-1][w]

        #2. 이동 함
        if w > 0:
            dp[t][w] = max(dp[t][w], dp[t-1][w-1])

        # 현재 위치
        pos = 1 if w%2 == 0 else 2

        if pos == info[t-1]:
            dp[t][w] += 1

print(max(dp[T]))
```
