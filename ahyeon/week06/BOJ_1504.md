### 📖 풀이한 문제

- 백준 1504 - 특정한 최단 경로
- 문제 링크: https://www.acmicpc.net/problem/1504
- 난이도: gold4
- 걸린 시간: 20m

### 🧩 풀이 설계 / 막힌 부분

- 처음 접근 아이디어: 다익스트라
- 고려한 시간복잡도: O(ElogN)
- 막힌 부분 / 해결 방법:
  - 막힌 건 없고 다익스트라를 구현해서 여러번 사용하면 되는 문제였다.
  - 정점을 두개 무조건 지나서 가야하기 때문에 1->v1->v2->n 혹은 1->v2->v1->n으로 가는 두 가지 루트를 고려해서 더 작은 것을 정답으로 뽑았다.

### ⭐️ 문제에서 주로 사용한 알고리즘

- 다익스트라

### 🧠 대략적인 코드 설명

- python

```
import sys
input = sys.stdin.readline
import heapq

n, e = map(int, input().split(" "))
graph = [[] for _ in range(n+1)]
for _ in range(e):
    a, b, c = map(int, input().split(" "))
    graph[a].append((c, b))
    graph[b].append((c, a))

v1, v2 = map(int, input().split(" "))

def dijkstra(s, e) :
    pq = []
    dist = [float("inf")] * (n+1)

    dist[s] = 0
    heapq.heappush(pq, (0, s))

    while len(pq) > 0:
        cost, now = heapq.heappop(pq)

        if cost > dist[now]:
            continue

        for c, next in graph[now]:
            new_cost = dist[now] + c

            if new_cost < dist[next]:
                dist[next] = new_cost
                heapq.heappush(pq, (new_cost, next))

    return dist[e]

route1 = dijkstra(1, v1) + dijkstra(v1, v2) + dijkstra(v2, n)
route2 = dijkstra(1, v2) + dijkstra(v2, v1) + dijkstra(v1, n)

if route1 >= float("inf") and route2 >= float("inf"):
    print(-1)
else:
    print(min(route1, route2))
```
