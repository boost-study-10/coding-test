### 🧩 풀이 설계 / 막힌 부분

- 처음 접근 아이디어: 십진수를 이진수로 변환 후 노드가 홀수개면 가운데 기준으로 나눠서 재귀탐색, 짝수면 가운데 두 수중 1인 값을 기준으로 나눠서 재귀탐색
- 고려한 시간복잡도:
- 막힌 부분 / 해결 방법:
  - 처음 접근한 아이디어가 완전히 틀렸다고 생각한다.
  - 문제에 나와있듯이 포화 이진트리가 되어야 한다.
  - 포화 이진 트리는 leaf가 아닌 내부 노드들은 모두 2개의 자식을 가지는 트리를 말한다.
  - 따라서 노드의 개수는 (2^k -1) 개가 되어야한다. 그러면 자연스럽게 가운데 노드가 생기고 양쪽 또한 3, 7 등과 같은 (2^k -1)개의 노드가 남는다
  - 따라서 현재 이진수에서 가장 가까운 2^k-1 값을 찾아 그 길이가 되도록 앞쪽에 0을 붙였다(앞에 0을 붙이면 실제 숫자엔 차이가 없다)
  - 그리고 확인할 것은 루트가 0이면 그 아래 서브트리에 1이 하나라도 있으면 안된다는 것이다. 이를 재귀를 통해 체크했다.

### ⭐️ 문제에서 주로 사용한 알고리즘

- dfs, 문자열, 이진변환

### 🧠 대략적인 코드 설명

- python
- fillzero: 2^k - 1 길이가 될 수 있도록 문자열 앞에 0붙이기
- dfs: 현재 구간(tree)에 대해 가운데 값이 '0'이라면 이 구간 안에 '1'이 하나라도 있는지 판단. 있으면 안됨

```
def fillzero(strBin):
    L = len(strBin)
    size = 1
    while size < L + 1:
        size *= 2
    return strBin.zfill(size-1)

def dfs(tree):
    if len(tree) == 1:
        return True

    mid = len(tree) // 2
    root = tree[mid]

    left = tree[:mid]
    right = tree[mid+1:]

    if root == '0':
        if '1' in left or '1' in right:
            return False

    return dfs(left) and dfs(right)

def solution(numbers):
    answer = []

    for n in numbers:
        strBin = bin(n)[2:]
        zeroBin = fillzero(strBin)
        answer.append(1 if dfs(zeroBin) else 0)
    return answer
```
