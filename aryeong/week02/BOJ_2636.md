### 📖 풀이한 문제
- 백준 2636 - 치즈
- 문제 링크: https://www.acmicpc.net/problem/2636
- 난이도: 골드4
- 걸린 시간: 40분

### 🧩 풀이 설계 / 막힌 부분
- 처음 접근 아이디어: 공기와 접촉한 치즈를 찾아야 한다. (0,0)에서 bfs로 공기 탐색. 만약에 1을 만나면 따로 저장해둠. 한 턴이 끝나면 arr에 녹인 치즈를 반영한다. 계속 반복. 만약에 치즈가 0개면 종료.
- 고려한 시간복잡도: O(nm * min(n,m)), n,m <=> 100
- 막힌 부분 / 해결 방법: map 전체에서 0인 좌표를 BFS 시작점으로 잡으려고 함. 이렇게 하면 내부 공기까지 외부 공기로 취급됨.

### ⭐️ 문제에서 주로 사용한 알고리즘
- bfs

### 🧠 대략적인 코드 설명
```js
const input = require("fs")
    .readFileSync("example.txt")
    .toString()
    .trim()
    .split("\n");

const [n, m] = input[0].split(' ').map(Number);
const arr = input.slice(1).map((line) => line.split(' ').map(Number));

const dy = [0, 1, 0, -1];
const dx = [1, 0, -1, 0];

function bfs() {
    const queue = [[0, 0]];
    let idx = 0;
    const visited = Array.from(Array(n), () => Array(m).fill(0));
    visited[0][0] = 1;

    const melt = [];

    while (idx < queue.length) {
        const [y, x] = queue[idx++];

        for (let i = 0; i < 4; i++) {
            const ny = y + dy[i];
            const nx = x + dx[i];
            if (ny < 0 || nx < 0 || ny >= n || nx >= m || visited[ny][nx]) continue;
            if (arr[ny][nx] === 0) queue.push([ny, nx]);
            if (arr[ny][nx] === 1) melt.push([ny, nx]);
            visited[ny][nx] = 1;
        }
    }

    for (const [y, x] of melt) {
        arr[y][x] = 0;
    }

    return melt.length;
}

let time = 0;
let last = 0;

while (true) {
    const melted = bfs();
    if (melted === 0) break;
    last = melted;
    time++;
}

console.log(time);
console.log(last);
```
