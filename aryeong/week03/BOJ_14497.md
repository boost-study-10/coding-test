### 📖 풀이한 문제
- 백준 14497 - 주난의 난
- 문제 링크: https://www.acmicpc.net/problem/14497
- 난이도: 골드4
- 걸린 시간: 50분

### 🧩 풀이 설계 / 막힌 부분
- 처음 접근 아이디어: 주난이가 목표 위치까지 이동하기 위해 최소 몇 번의 공격이 필요한지 구해야 함. 교실을 이동할 때 "0" : 빈 공간 -> 그대로 이동 가능(비용 0),"1" : 학생 -> 한 번 공격 후 이동(비용 1)
- 고려한 시간복잡도:
- 막힌 부분 / 해결 방법: 이동할 때 비용이 다르기 때문에 그냥 bfs로 해결할 수 없다. 비용 0은 queue 앞으로, 비용 1은 queue 뒤로 넣도록. deque로 풀어야 함.

### ⭐️ 문제에서 주로 사용한 알고리즘
- bfs

### 🧠 대략적인 코드 설명
```js
const input = require("fs")
  .readFileSync("/dev/stdin")
  .toString()
  .trim()
  .split("\n");

const [n, m] = input[0].split(" ").map(Number);
const [x1, y1, x2, y2] = input[1].split(" ").map(Number);
const classroom = input.slice(2).map((it) => it.trim().split(""));
const visited = new Array(n).fill(0).map(() => new Array(m).fill(0));
const queue = [[x1 - 1, y1 - 1, 0]];

const dy = [1, 0, -1, 0];
const dx = [0, 1, 0, -1];

let ret = Infinity;
while (queue.length) {
  queue.sort((a, b) => a[2] - b[2]);
  const [y, x, c] = queue.shift();

  if (y === x2 - 1 && x === y2 - 1) {
    ret = c;
    break;
  }

  for (let i = 0; i < 4; i++) {
    const nx = x + dx[i];
    const ny = y + dy[i];
    if (nx < 0 || ny < 0 || nx >= m || ny >= n || visited[ny][nx]) continue;
    visited[ny][nx] = 1;

    if (classroom[ny][nx] === "0") queue.unshift([ny, nx, c]);
    else queue.push([ny, nx, c + 1]);
  }
}

console.log(ret);
```
