### 📖 풀이한 문제
- 백준 13913 - 숨바꼭질 4
- 문제 링크: https://www.acmicpc.net/problem/13913
- 난이도: 골드4
- 걸린 시간: 40분

### 🧩 풀이 설계 / 막힌 부분
- 처음 접근 아이디어: dfs를 생각했으나 시간 복잡도에서 걸림. bfs로 푼다.
- 고려한 시간복잡도: dfs -> O(3^깊이), bfs -> O(n)
- 막힌 부분 / 해결 방법: visited 체크를 해줘야 같은 위치를 재방문 하지 않는다. 

### ⭐️ 문제에서 주로 사용한 알고리즘
- bfs

### 🧠 대략적인 코드 설명
```js
const input = require("fs")
    .readFileSync("/dev/stdin")
    .toString()
    .trim()
    .split("\n");

const [n, k] = input[0].split(' ').map(Number);

const visited = Array(100000 + 1).fill(false);
const prev = Array(100000 + 1).fill(-1);

const queue = [n];
visited[n] = true;
let idx = 0;

while(idx < queue.length) {
    const cur = queue[idx++];
    if(cur === k) break;

    const next = [cur - 1, cur + 1, cur * 2];

    for (const nx of next) {
        if (nx < 0 || nx > 100000) continue;
        if (visited[nx]) continue;

        visited[nx] = true;
        prev[nx] = cur;
        queue.push(nx);
    }
}

const path = [];
let cur = k;

while (cur !== -1) {
    path.push(cur);
    cur = prev[cur];
}

path.reverse();

console.log(path.length - 1);
console.log(path.join(' '));
```
