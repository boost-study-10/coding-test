### 📖 풀이한 문제

- 백준 12869 - 뮤탈리스크
- 문제 링크: https://www.acmicpc.net/problem/12869
- 난이도: 골드4
- 걸린 시간: 45분

### 🧩 풀이 설계 / 막힌 부분

- 처음 접근 아이디어: 그리디... 공격 순서에 따라 결과가 달라질 수 있어서 이렇게 접근할 수 없다.
- 고려한 시간복잡도: 61^3
- 막힌 부분 / 해결 방법: 체력 상태가 노드, 공격을 간선으로 생각 -> 최소 공격 횟수를 bfs로 찾는다. 공격 패턴은 (9,3,1)의 순열 6가지 가능. 각 상태에서 6개의 상태로 이동 가능.
=> 초기 상태에서 (0, 0, 0) 까지 최소 공격 횟수를 구한다.

### ⭐️ 문제에서 주로 사용한 알고리즘

- bfs

### 🧠 대략적인 코드 설명

```js
const input = require("fs")
  .readFileSync("example.txt")
  .toString()
  .trim()
  .split("\n");

const n = Number(input[0]);
const scv = input[1].split(" ").map(Number);
const attack = [
  [9, 3, 1],
  [9, 1, 3],
  [3, 9, 1],
  [3, 1, 9],
  [1, 9, 3],
  [1, 3, 9],
];

while (scv.length < 3) scv.push(0);

const visited = Array.from({ length: 61 }, () =>
  Array.from({ length: 61 }, () => Array(61).fill(-1)),
);

const queue = [[scv[0], scv[1], scv[2]]];
visited[scv[0]][scv[1]][scv[2]] = 0;

let idx = 0;

while (idx < queue.length) {
  const [x, y, z] = queue[idx++];

  if (x === 0 && y === 0 && z === 0) {
    console.log(visited[x][y][z]);
    break;
  }

  for (let i = 0; i < attack.length; i++) {
    na = Math.max(0, x - attack[i][0]);
    nb = Math.max(0, y - attack[i][1]);
    nc = Math.max(0, z - attack[i][2]);
    if (visited[na][nb][nc] >= 0) continue;
    visited[na][nb][nc] = visited[x][y][z] + 1;
    queue.push([na, nb, nc]);
  }
}
```
