### 📖 풀이한 문제
- 백준 16637 - 괄호 추가하기
- 문제 링크: https://www.acmicpc.net/problem/16637
- 난이도: 골드3
- 걸린 시간: ?

### 🧩 풀이 설계 / 막힌 부분
- 처음 접근 아이디어: 브루트포스로 dfs 사용하자. 괄호를 묶고/안 묶고 두 가지 갈래로 나눈다. 
- 고려한 시간복잡도: 2^9, 최대 연산자 개수가 9개
- 막힌 부분 / 해결 방법: 괄호 위치 조합을 만드려고 시도함. 구현이 복잡해서 왼쪽부터 계산 상태 유지하도록 접근. 
    1. 주어진 수식은 `숫자 연산자 숫자 연산자 ...` 구조. 
    2. dfs(다음에 계산할 연산자 인덱스, 지금까지 계산된 값) 
    3. - 괄호 안 묶고 -> 다음 연산자 이동 -> dfs(idx + 2)
        - 괄호 묶고 -> 괄호 계산한 연산자 건너뛰고 -> dfs(idx + 4)

### ⭐️ 문제에서 주로 사용한 알고리즘
- dfs

### 🧠 대략적인 코드 설명
```js
const input = require("fs")
    .readFileSync("/dev/stdin")
    .toString()
    .trim()
    .split("\n");

const n = Number(input[0]);
const arr = input[1].split('');

const calc = {
    '+': (a, b) => a + b,
    '-': (a, b) => a - b,
    '*': (a, b) => a * b,
}

let ans = -Infinity;

function dfs(idx, cur) {
    if (idx >= n) {
        ans = Math.max(ans, cur);
        return;
    }

    const next = calc[arr[idx]](cur, Number(arr[idx + 1]));
    dfs(idx + 2, next);

    if (idx + 3 < n) {
        const bracket = calc[arr[idx + 2]](Number(arr[idx + 1]), Number(arr[idx + 3]));
        const next2 = calc[arr[idx]](cur, bracket);
        dfs(idx + 4, next2);
    }
}

dfs(1, Number(arr[0]));
console.log(ans);
```
