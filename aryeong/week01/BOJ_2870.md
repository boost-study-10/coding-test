### 📖 풀이한 문제
- 백준 2870 - 수학숙제
- 문제 링크: https://www.acmicpc.net/problem/2870
- 난이도: 실버4
- 걸린 시간: 20분

### 🧩 풀이 설계 / 막힌 부분
- 처음 접근 아이디어: 1. line별로 정규식을 사용해서 숫자만 뽑는다. 2. 전체 ans 배열에 추가한다. 이때 0을 정리한다. 3. 오름차순으로 sort한 뒤 출력한다.
- 고려한 시간복잡도: O(M*NlogN) 인데 Max(M) = 50, Max(N) = 5000 (M은 문자열 최대 길이(숫자문자열), N은 sort 해야 하는 개수)
- 막힌 부분 / 해결 방법: 숫자가 50자리가 넘을 수 있는데 그러면 Number로 숫자를 다루기 어렵다 -> 문자열로 관리

### 🧠 대략적인 코드 설명
```js
const input = require("fs")
    .readFileSync("/dev/stdin")
    .toString()
    .trim()
    .split("\n");

const lines = input.slice(1).map((v) => v.trim());
const REGEXP = /[^0-9]+/g;
const ans = [];

for (const line of lines) {
    const nums = line.split(REGEXP).filter(Boolean);
    nums.forEach((num) => {
        const cleanNum = num.replace(/^0+/, "") || "0";
        ans.push(cleanNum);
    })
}

ans.sort((a, b) => {
    if (a.length !== b.length) return a.length - b.length;
    if (a < b) return -1;
    if (a > b) return 1;
});
console.log(ans.join('\n'));
```
