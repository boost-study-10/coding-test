### 📖 풀이한 문제
- 백준 2828 - 사과 담기 게임
- 문제 링크: https://www.acmicpc.net/problem/2828
- 난이도: 실버5
- 걸린 시간: 20분

### 🧩 풀이 설계 / 막힌 부분
- 처음 접근 아이디어: 바구니의 첫 칸과 마지막 칸에 현재 위치, 다음 위치를 담을 수 있도록 한다.
- 고려한 시간복잡도: O(n^2)
- 막힌 부분 / 해결 방법: 바구니 끝 위치 계산에서 -1을 안 뺌

### 🧠 대략적인 코드 설명
```js
const input = require("fs")
    .readFileSync("dev/stdin")
    .toString()
    .trim()
    .split("\n");

const [n, m] = input[0].split(' ').map(Number);
const j = Number(input[1]);
const apples = input.slice(2).map(Number);

let s = 1;
let move = 0;

for (let i = 0; i < j; i++) {
    const apple = apples[i];
    const e = s + m - 1;
    if (s <= apple && e >= apple) continue;
    if (s > apple) {
        move += s - apple;
        s = apple;
    } else {
        move += apple - e;
        s += apple - e;
    }
}

console.log(move);
```
