### 📖 풀이한 문제
- 백준 1992 - 쿼드트리
- 문제 링크: https://www.acmicpc.net/problem/1992
- 난이도: 실버 1
- 걸린 시간: 35분

### 🧩 풀이 설계 / 막힌 부분
- 처음 접근 아이디어: 재귀로 가자.. 1. 현재 영역이 하나의 값으로 표현 가능한지 2. 불가능하면 4등분해서 재귀적으로 처리
- 고려한 시간복잡도: n의 최대 크기가 64이니까 웬만해서 괜찮을거다. O(N^2 * logN)
- 막힌 부분 / 해결 방법: 쿼드트리 분할 기준을 어떻게 잡아야 하는지 GPT에게 물어봄

### ⭐️ 문제에서 주로 사용한 알고리즘
- 분할정복

### 🧠 대략적인 코드 설명
```js
const input = require("fs")
    .readFileSync("dev/stdin")
    .toString()
    .trim()
    .split("\n");

const n = Number(input[0]);
const video = input.slice(1).map((v) => v.trim().split(''));

function solve(y, x, size) {
    const first = video[y][x];

    for(let i = y;i < y + size;i++) {
        for(let j = x;j < x + size;j++) {
            if(video[i][j] !== first) {
                const half = size / 2;

                const topLeft = solve(y, x, half);
                const topRight = solve(y, x + half, half);
                const bottomLeft = solve(y + half, x, half);
                const bottomRight = solve(y + half, x + half, half);

                return "(" + topLeft + topRight + bottomLeft + bottomRight + ")";
            }
        }
    }

    return first;
}

console.log(solve(0, 0, n));
```
