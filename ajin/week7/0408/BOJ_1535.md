### 📖 풀이한 문제

- 백준 1535 - 안녕
- 문제 링크: https://www.acmicpc.net/problem/1535
- 난이도: 실버2
- 걸린 시간: 14분

### 🧩 풀이 설계 / 막힌 부분

- 문제: 사람들에게 인사할 때 체력이 줄고 기쁨이 늘어날 때, 체력이 0 이하가 되지 않도록 하면서 얻을 수 있는 최대 기쁨 구하기
- 처음 접근 아이디어:
  - 각 사람은 한 번만 인사할 수 있고, 인사할 때 체력 hp를 소모해서 기쁨 joy를 얻으므로 0/1 배낭 문제처럼 접근
  - dp[i]를 체력을 i만큼 사용했을 때 얻을 수 있는 최대 기쁨으로 두고 갱신하기
- 고려한 시간복잡도:
  - 사람 수 N <= 20이고 체력은 100일 때, 각 사람마다 체력 99까지 확인하므로 전체 시간복잡도는 O(N \* 100)이므로 O(N)
- 막힌 부분 / 해결 방법: X

### ⭐️ 문제에서 주로 사용한 알고리즘

- DP(배낭 문제)

### 🧠 대략적인 코드 설명

1. dp[j]를 체력을 j만큼 사용했을 때 얻을 수 있는 최대 기쁨으로 정의하고 0으로 초기화
2. 각 사람마다 체력 j를 99부터 hp까지 감소시키며 순회하면서
   - 현재 사람을 선택하지 않는 경우 dp[j]와,
   - 선택하는 경우 dp[j - hp] + joy를 비교해 더 큰 값으로 갱신
3. 체력을 최대 99까지 사용했을 때의 최대 기쁨 dp[99] 출력
