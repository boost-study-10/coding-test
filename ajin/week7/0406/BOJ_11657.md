### 📖 풀이한 문제

- 백준 11657 - 타임머신
- 문제 링크: https://www.acmicpc.net/problem/11657
- 난이도: 골드4
- 걸린 시간: 8분

### 🧩 풀이 설계 / 막힌 부분

- 문제:
  - 1번 도시에서 출발해 나머지 모든 도시까지 가는 최단 거리 구하기
  - 음수 간선이 존재할 수 있고, 1번 도시에서 도달 가능한 음수 사이클이 있으면 -1 출력하기
- 처음 접근 아이디어:
  - 음수 가중치가 있는 최단 거리 문제이므로 벨만포드 사용하기
- 고려한 시간복잡도:
  - N <= 500, M <= 6,000일 때, 벨만포드는 정점 수만큼 모든 간선을 확인하므로
  - O(N _ M) = 500 _ 6,000 = 3,000,00 충분히 가능
- 막힌 부분 / 해결 방법: X

### ⭐️ 문제에서 주로 사용한 알고리즘

- 최단 거리(벨만포드)

### 🧠 대략적인 코드 설명

1. 모든 버스 정보를 [A, B, C] 형태로 저장
2. 거리 배열 distance를 Infinity로 초기화하고, 시작 도시 1번의 거리를 0으로 설정
3. N번 반복하면서 모든 간선을 순회
   > N - 1번 동안 최단 거리 업데이트, 마지막 N번 째에서 음수 사이클 확인
   - 간선 A -> B에 대해 distance[A]가 아직 Infinity면 시작점에서 도달할 수 없는 정점이므로 건너뜀
   - distance[A] + C가 현재 distance[B]보다 작으면 거리 갱신
   - 만약 N번째 반복에서도 값이 갱신되면 -1 반환
4. 음수 사이클이 없으면 2번 도시부터 N번 도시까지의 최단 거리 출력
   - 끝까지 도달할 수 없는 도시(거리가 Infinity)는 -1로 변환해서 출력
