### 📖 풀이한 문제

- 백준 1106 - 호텔
- 문제 링크: https://www.acmicpc.net/problem/1106
- 난이도: 골드4
- 걸린 시간: 17분

### 🧩 풀이 설계 / 막힌 부분

- 문제: 각 도시에서 특정 비용으로 고객을 유치할 수 있는 광고를 할 때, 적어도 C명 이상의 고객을 확보하기 위한 최소 비용 구하기
- 처음 접근 아이디어:
  - 도시마다 광고를 여러 번 할 수 있으므로, 여러 번 사용할 수 있는 배낭 문제로 접근
  - dp[i]를 고객 i명을 확보하는 데 필요한 최소 비용으로 두고 갱신하기
  - 적어도 C명 이상을 만족하면 되기 때문에 i < customer인 경우도 DP[i]와 cost를 비교
- 고려한 시간복잡도:
  - C <= 1000, 도시 수 N <= 20 일 때,
  - 각 도시에 대해 고객 수 0 ~ C까지 순회하므로 O(N \* C)
- 막힌 부분 / 해결 방법: X

### ⭐️ 문제에서 주로 사용한 알고리즘

- DP

### 🧠 대략적인 코드 설명

1. dp[i]를 고객 i명을 확보하는 데 필요한 최소 비용이라고 두고, 모두 Infinity로 초기화
   - dp[0]은 0으로 설정
2. 각 도시의 광고 정보 [cost, customer]를 하나씩 확인하면서
   1. i < customer일 때, 최소 비용 설정하기
      - dp[i] = Math.min(dp[i], cost)
   2. customer <= i <= C일 때, 현재 도시 광고를 여러 번 사용했을 때 최솟값 업데이트
      - dp[i] = Math.min(dp[i], dp[i - customer] + cost)
3. dp[C] 출력
