### 📖 풀이한 문제

- 백준 1904 - 01타일
- 문제 링크: https://www.acmicpc.net/problem/1904
- 난이도: 실버3
- 걸린 시간: 15분 / 2트

### 🧩 풀이 설계 / 막힌 부분

- 문제: 길이가 N인 수열을 1과 00만 사용해서 만들 수 있을 때, 가능한 모든 수열의 개수를 15746으로 나눈 나머지 구하기
- 처음 접근 아이디어:
  - 길이가 i인 수열의 마지막은 1로 끝나거나 00으로 끝나는 두 경우로 나눌 수 있음
  - 마지막이 1이면 앞부분은 길이 i - 1인 경우의 수이고, 마지막이 00이면 앞부분은 길이 i - 2인 경우의 수이므로
  - dp[i] = dp[i - 1] + dp[i - 2] 점화식으로 정의 가능
- 고려한 시간복잡도: N <= 1,000,000일 때, 3부터 N까지 한 번씩만 순회하며 값을 구하므로 O(N)
- 막힌 부분 / 해결 방법:
  - 15746 나누기 조건 놓쳐서 한 번 틀림

### ⭐️ 문제에서 주로 사용한 알고리즘

- DP

### 🧠 대략적인 코드 설명

1. dp[i]를 길이가 i인 01타일 수열의 개수로 정의하고 N + 1 크기의 배열 생성
2. 길이 1은 한 가지(1), 길이 2는 두 가지(00, 11)이므로 dp[1] = 1, dp[2] = 2로 초기화
3. 3부터 N까지 순회하면서 dp[i] = (dp[i - 1] + dp[i - 2]) % 15746으로 갱신
4. dp[N] 출력
