### 📖 풀이한 문제

- 백준 2193 - 이친수
- 문제 링크: https://www.acmicpc.net/problem/2193
- 난이도: 실버3
- 걸린 시간: 9분 / 2트

### 🧩 풀이 설계 / 막힌 부분

- 문제: 길이가 N인 이친수의 개수 구하기
- 처음 접근 아이디어:
  - 이친수는 0으로 시작하면 안 되고, 1이 두 번 연속 나오면 안 될 때,
  - 마지막 자리가 0인지, 1인지 나누면 이전의 경우의 합으로 표현 가능
  - 0으로 끝나는 경우: 길이 i - 1인 이친수에 0을 붙이기 -> dp[i - 1]
  - 1로 끝나는 경우: 마지막이 1이면 그 앞자리가 0이므로 ~01 형태 -> dp[i - 2]
- 고려한 시간복잡도:
  - 4부터 N까지 한 번씩만 확인하면서 dp를 구하므로 O(N)
- 막힌 부분 / 해결 방법:
  - 경우의 수를 잘못 구해 틀림
  - 직접 경우의 수를 확인하는 경우에는 규칙이 맞는지 점검하기

### ⭐️ 문제에서 주로 사용한 알고리즘

- DP

### 🧠 대략적인 코드 설명

1. dp[i]를 길이가 i인 이친수의 개수로 정의하고 배열 생성
2. dp[1] = 1, dp[2] = 1로 초기화
3. 3부터 N까지 순회하면서 dp[i] = dp[i - 1] + dp[i - 2]로 갱신
4. dp[N] 출력
