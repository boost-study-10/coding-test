### 📖 풀이한 문제

- 백준 2133 - 타일 채우기
- 문제 링크: https://www.acmicpc.net/problem/2133
- 난이도: 골드4
- 걸린 시간: 25분

### 🧩 풀이 설계 / 막힌 부분

- 문제: 3×N 크기의 벽을 2×1, 1×2 타일로 빈칸 없이 채우는 경우의 수 구하기
- 처음 접근 아이디어: 직접 그려서 점화식 구하기
  - N이 홀수이면 3×N을 타일로 모두 채울 수 없으므로 경우의 수는 0
  - N이 짝수일 때만 기본이 되는 3×2 모양은 3가지가 있고, 그 외에는 이전 경우에 특수한 모양이 추가되는 형태
- 고려한 시간복잡도:
  - N <= 30 일 때, 각 i마다 이전 짝수 길이들을 다시 확인하므로 O(N²)
- 막힌 부분 / 해결 방법: X

### ⭐️ 문제에서 주로 사용한 알고리즘

- DP(도형 점화식)

### 🧠 대략적인 코드 설명

1. dp[i]를 3×i 벽을 채우는 경우의 수로 정의하고 배열 초기화
2. 아무 타일도 놓지 않는 경우를 기준으로 dp[0] = 1, 기본 모양 3가지를 반영해 dp[2] = 3으로 설정
3. i를 4부터 N까지 2씩 증가시키며 짝수 너비만 계산하기
   1. 먼저 마지막 2칸을 기본 3가지 방식으로 채우는 경우를 반영해서 dp[i] = dp[i - 2] \* 3
   2. 그다음 j를 4부터 i까지 2씩 증가시키며 이전 길이 i - j 뒤에 특수한 모양을 붙이는 경우를 dp[i - j] \* 2씩 더하기
4. dp[N] 출력
