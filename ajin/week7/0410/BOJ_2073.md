### 📖 풀이한 문제

- 백준 2073 - 수도배관공사
- 문제 링크: https://www.acmicpc.net/problem/2073
- 난이도: 골드4
- 걸린 시간: 25분

### 🧩 풀이 설계 / 막힌 부분

- 문제:
  - 길이와 용량이 주어진 파이프를 이어 총 길이 D를 만들 때,
  - 연결한 배관의 용량을 사용한 파이프들 중 최솟값으로 한다면
  - 연결된 배관의 최대 용량 구하기
- 처음 접근 아이디어:
  - 파이프를 여러 개 이어 붙여 길이 D를 만들어야 하고, 최종 용량은 선택한 파이프들의 용량 중 최솟값이 되므로
  - dp[l]를 길이 l을 만들었을 때 가능한 최대 용량으로 두고 배낭 문제처럼 접근
- 고려한 시간복잡도:
  - P <= 350, D <= 100,000 일 때,
  - 파이프 개수 P개에 대해 각 파이프마다 길이 D부터 L까지 확인하므로 O(P × D) = 35,000,000
- 막힌 부분 / 해결 방법: X

### ⭐️ 문제에서 주로 사용한 알고리즘

- DP(배낭 문제?)

### 🧠 대략적인 코드 설명

1. dp[l]를 길이 l을 만들었을 때 가능한 최대 용량으로 정의
   - 길이 0은 아직 아무 파이프도 사용하지 않은 상태이므로 Infinity,
   - 나머지는 만들 수 없는 상태이므로 0으로 초기화
2. 각 파이프마다 길이 D부터 L까지 역순으로 순회하면서
   - 기존 값 dp[l]과 현재 파이프를 포함한 경우를 비교해 더 큰 값으로 갱신
   - 현재 파이프를 포함한 경우의 용량은 기존 길이 l - L을 만들던 배관의 용량과 현재 파이프 용량 C 중 작은 값
3. 모든 파이프를 확인한 뒤 dp[D] 출력
