### 📖 풀이한 문제

- 백준 11047 - 동전 0
- 문제 링크: https://www.acmicpc.net/problem/11047
- 난이도: 실버4
- 걸린 시간: 8분

### ⭐️ 문제에서 주로 사용한 알고리즘

- 그리디

### 🧩 풀이 설계 / 코드 설명

#### 시간 복잡도

1. 부르트포스 → 불가능
   - N = 10, K = 100,000,000일 때, 모든 조합/백트래킹으로 탐색하면 완전 불가능
2. 그리디 → 가능
   - 동전이 배수 관계를 만족하기 때문에 그리디 가능
   - 높은 가치인 동전을 최대한 많이 사용하면서 K 만들기
   - O(N)
3. DP → 가능하지만 쉬운 방법이 있으니 패스

#### 설계

1. 현재 금액을 money = K로 설정
2. 가치가 큰 동전부터 순회
3. (현재 동전 가치 <= money) 라면
   - money / coin 해서 최대 개수 구하고 money 업데이트
   - 정답에 개수 추가
4. money가 0이 되면 종료
