### 📖 풀이한 문제

- 백준 1449 - 수리공 항승
- 문제 링크: https://www.acmicpc.net/problem/1449
- 난이도: 실버3
- 걸린 시간: 15분

### ⭐️ 문제에서 주로 사용한 알고리즘

- 그리디

### 🧩 풀이 설계 / 코드 설명

#### 시간 복잡도

1. ~부르트포스 → 불가능~
   - n = 1000, l = 1000 일 때, 모든 물 새는 위치에 테이프 붙이기
   - 최소 조건 불만족 및 O(2^n)으로 불가능
2. 그리디 → 가능
   - 위치 정렬 하고 앞에서부터 테이프 붙이기
   - O(n log n) + O(n)

#### 설계

1. 물 새는 곳 정렬하기
2. 앞에서부터 테이프 붙이기
   - 해당 위치에 테이프가 붙어 있으면 넘기기
   - 테이프가 없으면 해당 위치부터 새로운 테이프 붙이고
     테이프 종료 위치 저장하고 개수 증가하기

#### 왜 그리디가 성립하는지

> 항상 앞쪽 위치부터 테이프를 붙이기가 최선인지 검증

- 앞쪽이 아닌 다른 위치에 테이프를 먼저 붙인 것이 최적의 해라면,
- 결국.. 또.. 가장 앞에 있는 위치는 별도의 테이프로 막아야함
