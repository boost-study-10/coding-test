### 📖 풀이한 문제

- 백준 1697 - 숨바꼭질
- 문제 링크: https://www.acmicpc.net/problem/1697
- 난이도: 실버1
- 걸린 시간: 40분 / 2트

### 🧩 풀이 설계 / 막힌 부분

- 처음 접근 아이디어: x에서 -1, +1, +x 만큼 이동 가능한 그래프의 최단 거리 문제
- 고려한 시간복잡도:
  - 위치 범위가 0 ~ 100,000이고 방문할 때마다 다음 위치 3개 확인
  - BFS로 탐색한다면 O(V + E)이고 O(100,000 + 3)..?
- 막힌 부분 / 해결 방법:
  - 예외조건 처리 안해서 1트 실패..
  - 문제 꼼꼼하게 읽기!!!

### ⭐️ 문제에서 주로 사용한 알고리즘

- BFS
  - 가중치가 모두 동일한 최단 경로 문제
  - DFS는 모든 경로를 탐색하고 비교해야하지만,
  - BFS는 처음 도착점을 만났을 때가 최단 거리이므로 유리함

### 🧠 대략적인 코드 설명

1. `x`에 도달하는 데 걸린 최소 시간을 visited로 정의
2. 시작 위치(n)에서 BFS 돌기
   1. 현재 위치(x)에서 갈 수 있는 다음 위치 계산
      - `x - 1`, `x + 1`, `x + x` (`2 * x`)
   2. 다음 위치가 범위를 벗어나지 않고, 아직 방문하지 않았다면
      - 큐에 넣고 `visited[next] = visited[current] + 1`
3. 목표 위치(k)에 도달하면 탐색을 종료하고 `visited[k]`를 출력
