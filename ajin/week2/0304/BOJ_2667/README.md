### 📖 풀이한 문제

- 백준 2667 - 단지번호붙이기
- 문제 링크: https://www.acmicpc.net/problem/2667
- 난이도: 실버1
- 걸린 시간: 22분 / 1트

### 🧩 풀이 설계 / 막힌 부분

- 처음 접근 아이디어: 기본적인 그래프 탐색 문제
- 고려한 시간복잡도:
  - N = 25 일 때, 최대 25 \* 25 = 625 칸
  - DFS/BFS로 탐색하면 O(N^2) 최악의 경웨도 625칸만 조회
- 막힌 부분 / 해결 방법: X

### ⭐️ 문제에서 주로 사용한 알고리즘

- DFS
  - 그래프 방문 문제이므로 DFS/BFS 모두 가능

### 🧠 대략적인 코드 설명

1. 지도 순회하면서 집(1) 찾기
   - 현재 칸을 방문 처리(`0`)하고
   - 해당 칸에서 DFS를 돌며 연결된 집 수 증가하기
2. 단지/집 수
   - `house.length` 출력 → 총 단지 수
   - `house`를 오름차순 정렬 후 출력 → 각 단지 내 집 수
