### 📖 풀이한 문제

- 백준 10026 - 적록색약
- 문제 링크: https://www.acmicpc.net/problem/10026
- 난이도: 골드5
- 걸린 시간: 20분 / 1트

### 🧩 풀이 설계 / 막힌 부분

- 처음 접근 아이디어: R/G/B/R+G 그래프 탐색하기
- 고려한 시간복잡도:
  - N = 100 일 때, 최대 100 \* 100 = 10,000 칸
  - 각 칸은 일반 시야 탐색 1번, 적록색약 탐색에서 1번 방문
  - O(N^2) = 10,000
- 막힌 부분 / 해결 방법: X

### ⭐️ 문제에서 주로 사용한 알고리즘

- DFS
  - 그래프 방문 문제이므로 DFS/BFS 모두 가능

### 🧠 대략적인 코드 설명

1. 일반 시야용 방문 배열 `visited`, 적록색약용 방문 배열 `red_green_visited` 따로 처리
2. 전체 칸을 순회하면서
   - 일반 시야에서 아직 방문하지 않은 칸이면 탐색 시작
   - 적록색약 시야에서 아직 방문하지 않았고 현재 칸이 `B`가 아니면 탐색 시작
     - `B`는 적록색약 유무와 상관 없으므로 일반 시야에서 먼저 수행됨
   - 일반 시야 DFS와 적록색약 DFS를 각각 수행
