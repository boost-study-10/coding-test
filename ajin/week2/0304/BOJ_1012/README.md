### 📖 풀이한 문제

- 백준 1012 - 유기농 배추
- 문제 링크: https://www.acmicpc.net/problem/1012
- 난이도: 실버2
- 걸린 시간: 20분 / 1트

### 🧩 풀이 설계 / 막힌 부분

- 처음 접근 아이디어: 그래프의 모든 점을 방문해, 그래프 탐색 횟수 구하기
- 고려한 시간복잡도:
  - M = 50, N = 50, K = 2500일 때,
  - 전체 칸을 한 번씩 보고 각 칸은 최대 한 번만 DFS로 방문되므로 O(N × M)
- 막힌 부분 / 해결 방법:
  - 지문에서 제시하는 (x, y)를 확인하지 않아 반대로 입력받음ㅠ.ㅠ
  - 디버깅으로 찾아서 해결..

### ⭐️ 문제에서 주로 사용한 알고리즘

- DFS
  - 그래프 연결 요소를 구하는 문제라, BFS/DFS 둘 다 가능

### 🧠 대략적인 코드 설명

1. 밭을 순회하면서 값이 지렁이(1)인 칸 찾기
2. 아직 방문하지 않은 배추라면,
   1. 지렁이 수(DFS를 시작한 횟수) 1 증가
   2. 현재 칸을 방문 처리(2)
   3. DFS를 시작해서 연결된 모든 배추를 방문 처리
3. 밭 전체 순회가 끝나면 지렁이 수 출력
