### 📖 풀이한 문제

- 백준 2606 - 바이러스
- 문제 링크: https://www.acmicpc.net/problem/2606
- 난이도: 실버3
- 걸린 시간: 10분 / 2트

### 🧩 풀이 설계 / 막힌 부분

- 처음 접근 아이디어: 1번 컴퓨터와 연결된 컴퓨터 개수 구하기, 기본적인 그래프 탐색 문제
- 고려한 시간복잡도:
  - 컴퓨터 수 100일 때, 인접 리스트로 탐색하면 O(V + E) = O(200) 정도..
- 막힌 부분 / 해결 방법: X

### ⭐️ 문제에서 주로 사용한 알고리즘

- DFS
  - 특정 노드와 연결된 모든 노드 찾는 문제이므로 DFS/BFS 모두 가능

### 🧠 대략적인 코드 설명

1. 인접 리스트로 그래프 저장하기
2. 1번 컴퓨터에서 DFS 탐색하기
   1. 새로운 컴퓨터면 감염 컴퓨터 수 증가하기
   2. 다음 컴퓨터로 이동하기
