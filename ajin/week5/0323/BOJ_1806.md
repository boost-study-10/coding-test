### 📖 풀이한 문제

- 백준 1806 - 부분합
- 문제 링크: https://www.acmicpc.net/problem/1806
- 난이도: 골드4
- 걸린 시간: 16분 / 3트

### 🧩 풀이 설계 / 막힌 부분

- 문제: 길이 N인 수열에서 연속된 부분수열의 합이 S 이상이 되는 구간 중 길이의 최솟값 구하기
- 처음 접근 아이디어:
  - 연속된 구간의 합을 구해야 하므로 투포인터 사용
  - left와 right로 현재 구간을 잡고, 구간합 sum을 관리
- 고려한 시간복잡도:
  - N이 100,000이므로 모든 시작점마다 끝점을 다시 찾는 O(N^2)은 불가능
  - 투포인터를 사용하면 left, right가 각각 한 번씩만 증가하므로 O(N)에 해결 가능
- 막힌 부분 / 해결 방법:
  - `이 수열에서 연속된 수들의 부분합 중에 그 합이 S 이상이 되는 것 중` 을 서로 다른 수라 생각해 틀림
  - 반례 체크 왜 안해??? 특히 경계값 중심으로!!!

### ⭐️ 문제에서 주로 사용한 알고리즘

- 투포인터

### 🧠 대략적인 코드 설명

1. 최소 길이 정답을 N + 1로 초기화
2. `left = 0`, `right = 0`으로 시작하고 현재 구간합 `sum = arr[0]`으로 설정
   - 현재 구간합 sum이 S보다 작으면
     - 조건을 만족하지 못하므로 right를 늘려 더 큰 구간을 만들기
     - 새로 포함된 값을 sum에 더하기
   - 현재 구간합 sum이 S 이상이면
     - 현재 구간 길이 `right - left + 1`로 정답을 갱신
     - 더 짧은 구간이 가능한지 확인하기 위해 arr[left]를 빼고 left++
3. 탐색이 끝난 뒤 정답이 갱신되지 않았다면 0, 아니면 최소 길이를 출력
