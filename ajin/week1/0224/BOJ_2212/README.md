### 📖 풀이한 문제

- 백준 2212 - 센서
- 문제 링크: https://www.acmicpc.net/problem/2212
- 난이도: 골드5
- 걸린 시간: 15분

### ⭐️ 문제에서 주로 사용한 알고리즘

- 그리디

### 🧩 풀이 설계 / 코드 설명

#### 시간 복잡도

1. 부르트포스 → 불가능

- N = 10000, K = 1000일 때, N-1 중에서 K-1를 선택해야함 -> 절대 불가능..

2. 그리디 → 가능

- (1)센서를 정렬하고 (2)센서 간의 거리를 구한 후 (3)구간 중 작은 것의 (4)합 구하기
- O(N logN) + O(N) + O(N logN) + O(N) = O(N logN)

#### 설계

- 센서 구간을 구한 후 특정 구간을 제거하면 센서를 특정 그룹으로 나누는 것과 같음
- 구간을 제거할수록 집중국 수신 가능 영역의 길이는 줄어듦
- 센서가 N개면 인접한 구간은 N-1개이고, K개의 그룹을 만들기 위해 K-1개의 구간을 제거해야 함

1. 센서 정렬
2. 센서 간 거리 구하기
3. 구간을 오름차순 정렬해서
4. 가장 긴 구간 K-1개 제거하고
5. 남은 구간의 합 구하기
