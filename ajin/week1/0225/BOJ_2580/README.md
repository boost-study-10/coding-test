### 📖 풀이한 문제

- 백준 2580 - 스도쿠
- 문제 링크: https://www.acmicpc.net/problem/2580
- 난이도: 골드4
- 걸린 시간: 40분

### ⭐️ 문제에서 주로 사용한 알고리즘

- 백트래킹

### 🧩 풀이 설계 / 코드 설명

#### 시간 복잡도

1. 부르트포스
   - 빈 칸 개수 K = 81 일 때, 9^81.. 절대 불가능
   - 단, 한 칸에 넣는 숫자를 제한한다면 가능..할지도...(백트래킹)

#### 설계

1. 게임 시작 전 보드를 순회하면서 (i, j)에 대해
   - 숫자가 있는 칸: 가로[i], 세로[j], 정사각[(i / 3) * 3 + (j / 3)]에 숫자 저장
   - 숫자가 없는 칸: blank에 (i, j) 저장
2. 숫자 없는 칸을 돌면서 백트래킹으로 숫자 채우기
   - 종료: blank의 모든 칸을 채우기
   - 1~9 숫자 중에서 이미 가로/세로/정사각형에 해당 숫자가 존재하면 넘어가기
   - 없다면 보드에 숫자 채우고 가로/세로/정사각형에 숫자 추가하기
   - 막혀서 돌아왓다면 보드 원상복구하고 숫자도 제거
3. 모든 숫자 없는 칸을 채우면 바로 종료하기

#### 가로, 세로, 정사각 배열

- (i,j)에 대해
- row[i] : i번째 행에 이미 사용된 숫자 집합
- col[j] : j번째 열에 이미 사용된 숫자 집합
- square[s] : s번째 3×3 구역에 이미 사용된 숫자 집합
  - s: Math.floor(i / 3) \* 3 + Math.floor(j / 3)

#### 백트래킹 중 바로 종료하기

```js
function backtracking(index) {
  if (특정 조건) {
    return true;
  }

  for (...) {
   // push: 선택 적용

    if (backtracking(index + 1)) return true;

   // pop: 선택 취소
  }

  return false;
}
```

- false, true 조건이 없다면 답은 찾은 후에도 불필요한 탐색을 진행함

```
level 0
 └─ level 1
     └─ level 2
         └─ ...
             └─ 정답 발견 → true 반환
         ← true 전파
     ← true 전파
← true 전파
```

1. 특정 조건으로 정답을 찾았다면, 부모로 true 전달
2. 부모에서 받으면, 즉시 상위 호출로 true 전달
3. 최상위까지 true가 전파되어 재귀 종료됨

- 모든 숫자를 시도해도 조건을 만족하지 못하면 false로 실패 처리
