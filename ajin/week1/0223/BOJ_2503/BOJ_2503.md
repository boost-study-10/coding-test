### 📖 풀이한 문제

- 백준 2503 - 숫자야구
- 문제 링크: https://www.acmicpc.net/problem/2503
- 난이도: 실버3
- 걸린 시간: 35분

### ⭐️ 문제에서 주로 사용한 알고리즘

- 브루트포스

### 🧩 풀이 설계 / 코드 설명

#### 시간 복잡도

1. 부르트포스 → 가능
   - 최악의 경우: n = 100 일 때, 1~9로 만들 수 있는 세 자리 수를 모두 만들고(9P3) 질문에 일치하는지 체크하기
   - O(n \* 9 \* 8 \* 7) = 약 50,000
2. 생략...

#### 설계

1. 1~9로 구성된 서로 다른 세 자리 수 만들기
2. 세자리 수가 입력으로 주어진 질문의 답의 스트라이크/볼과 일치하는 지 확인하기
3. n개의 답변 모두 일치하다면 답의 개수 1 증가하기

#### 스트라이크/볼 계산

- 스트라이크: 같은 숫자가 같은 위치에 있음
  - 값이 같고 인덱스도 같음
- 볼: 같은 숫자가 있지만 위치는 다름
  - 값은 값고 인덱스는 다름

#### 원트 실패..

1. 세 자리 수가 중복될 수 없다는 조건 놓침
2. 구현할 때, 스트라이크는 볼과 중복할 수 없다는 점 놓침
   - 3 스트라이크는 항상 0볼이다

### 기억해!!!!

#### permutation도 30초 안에 만들어야함

```js
/**
 *
 * @param arr 순열 구할 배열
 * @param cur 현재 만들고 있는 순열 배열
 * @returns
 */
function permutation(arr, cur) {
  if (cur.length === R) {
    if (check_question(cur)) answer++;
    return;
  }

  for (let i = 0; i < arr.length; i++) {
    if (checked[i]) continue;

    cur.push(arr[i]);
    checked[i] = true;

    permutation(arr, cur);

    cur.pop();
    checked[i] = false;
  }
}
```
