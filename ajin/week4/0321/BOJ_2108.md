### 🧩 풀이 설계 / 막힌 부분

- 문제: N 개의 수에 대해 산술평균, 중앙값, 최빈값, 범위 구하기
- 처음 접근 아이디어:
  - 설계없이 요구하는 값 바로 구형
- 고려한 시간복잡도:
  - N === 500,000 일 때, ?
- 막힌 부분 / 해결 방법:

1. 산술평균을 반올림할 때 -0이 나오는 경우 처리하기
   - (0 + 0 + (-1)) / 3 = -0.333... 이고 이를 반올림하면 JS에서는 -0로 나옴
   - -0은 부동소수점!!에서 부호 정보를 위해 존재하는 값으로 0과 -0은 같은 값으로 취급됨
   - 하지만, 문제의 출력은 0이어야 하므로 if (answer[0] === -0) answer[0] = 0 로 보정해서 해결
2. 틀린 부분: 소수점 이하 첫째 자리에서 반올림한 정수를 구하기
   - Math.round((sum / n).toFixed(1)) -> Math.round(sum / n)
   - toFixed(1)은 숫자를 반올림한 뒤 문자열로 바꾸는 메서드라, 이미 한 번 반올림이 일어난 값을 다시 Math.round 하게 됨
   - 그리고 Math.round() 자체가 소수점 첫째 자리 기준으로 반올림함!!
