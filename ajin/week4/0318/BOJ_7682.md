### 📖 풀이한 문제

- 백준 7682 - 틱택토
- 문제 링크: https://www.acmicpc.net/problem/7682
- 난이도: 골드4
- 걸린 시간: 33분 / 2트

### 🧩 풀이 설계 / 막힌 부분

- 처음 접근 아이디어:
  - 문자열로 주어진 게임판을 3×3 배열로 변환
  - X와 O의 개수를 세어 기본적으로 가능한 턴 수인지 확인
  - X 또는 O가 이긴 상태인지 확인
  - 승자가 없다면, 판이 가득 찬 무승부 상태인지 확인
- 고려한 시간복잡도:
  - 게임판이 3X3으로 모든 연산은 상수 시간?
- 막힌 부분 / 해결 방법
  1. 기본 구현에서 시간이 오래 걸림
     - 배열에서 특정 위치 값을 비교하기
     - 배열 요소 개수를 세기
     - 한 줄의 값이 모두 같은지 검사하기
  2. 틀린 이유
     - `X`와 `O`가 동시에 이긴 상태는 불가능하다는 조건을 놓침
  3. 추가로
     - 3×3 판이어서 2차원 배열로 바꿔도 충분히 구현 가능하지만,
     - 실제로는 칸 수가 9칸뿐이므로 문자열 그대로 처리해도 훨씬 간단하게 구현 가능

### 🧠 대략적인 코드 설명

1. 입력으로 들어온 9글자 문자열을 3×3 게임판으로 변환
2. 가로, 세로, 대각선의 8개 승리 경우를 검사해서 누가 이겼는지 확인
3. 승리한 줄이 하나도 없으면, 판이 가득 찬 상태인지 확인
4. 승리한 플레이어가 있다면 `X`, `O` 개수가 그 결과와 맞는지 확인
5. X`와 `O`가 동시에 이긴 경우는 바로 invalid 처리
6. 모든 조건을 통과하면 valid, 아니면 invalid를 출력
