### 📖 풀이한 문제

- 백준 7576 - 토마토
- 문제 링크: https://www.acmicpc.net/problem/7576
- 난이도: 골드5
- 걸린 시간: 20분 / 1트

### 🧩 풀이 설계 / 막힌 부분

- 처음 접근 아이디어: 시작점이 여러 개인 BFS
- 고려한 시간복잡도:
  - M = 1000, N = 1000이면 BFS해도 O(M\*N) = 1,000,000
- 막힌 부분 / 해결 방법: x

### ⭐️ 문제에서 주로 사용한 알고리즘

- BFS

### 🧠 대략적인 코드 설명

1. 박스를 모두 보면서 익은 토마토(1) 위치 저장
2. 익은 토마토를 시작점으로 모두 넣은 뒤 BFS 돌기
   - 덜 익은 토마토(0)라면 day + 1해서 저장
   - BFS 종료 시점의 day가 최대 요일
3. 박스를 모두 보면서 덜 익은 토마토가 있으면 -1 반환
4. 모두 익은 토마토라면 최대 요일 반환
