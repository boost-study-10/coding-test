### 📖 풀이한 문제

- 백준 2206 - 벽 부수고 이동하기
- 문제 링크: https://www.acmicpc.net/problem/2206
- 난이도: 골드3
- 걸린 시간: 60분 / 2트 / 정답 확인

### 🧩 풀이 설계 / 막힌 부분

- 처음 접근 아이디어:
  - 모든 벽 한 번만 부술 수 있으니, 벽 부수는 모든 경우에 대해 BFS돌아서 최단 거리 찾기
  - 메모리, 시간 초과
- 고려한 시간복잡도:
  - N=1000, M = 1000으로 당연히 시간 초과하지만 방법이 생각나지 않음..
- 막힌 부분 / 해결 방법
  - 벽을 부쉈는지 여부를 관리하며 BFS는 1번만 돌아야 함
  - 같은 좌표여도 벽을 부수고 온 경우 / 안 부수고 온 경우를 다르게 취급해야 함

### ⭐️ 문제에서 주로 사용한 알고리즘

- BFS

### 🧠 대략적인 코드 설명

1. visited[x][y][broken]로 각 좌표에서 벽 부순 여부도 저장
   - broken = 0이면 벽을 부수지 않은 상태
   - broken = 1이면 벽을 부순 상태
2. 시작점(0,0)에서 BFS를 돌면서
   1. 다음 칸이 빈 칸이라면 broken 상태 유지한 채 이동
   2. 다음 칸이 벽이고 아직 부수지 않았다면, 벽을 부수고 이동
3. 도착점(n-1,m-1)이면 거리 반환
4. BFS 다 돌았다면(도착점 도달 못하면) -1 반환
