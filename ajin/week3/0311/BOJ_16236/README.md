### 📖 풀이한 문제

- 백준 16236 - 아기상어
- 문제 링크: https://www.acmicpc.net/problem/16236
- 난이도: 골드3
- 걸린 시간: 45분 / 1트

### 🧩 풀이 설계 / 막힌 부분

- 처음 접근 아이디어:
  - 현재 상어 위치에서 BFS 돌면서 먹을 수 있는 물고기 위치 구하기
  - 해당 물고기 잡아먹고 상어 상태 업데이트하기
- 고려한 시간복잡도:
  - N = 20일 때, 400 칸에서 물고기 선택하려고 BFS 돌면 O(N^2)
  - 물고기 선택하는 과정을 최대 400칸에서 수행가능..?: O(N^2 \* N^2) = 160,000
- 막힌 부분 / 해결 방법: x

### ⭐️ 문제에서 주로 사용한 알고리즘

- BFS
- 구현

### 🧠 대략적인 코드 설명

1. 아기 상어의 위치, 크기, 먹은 물고기 수는 객체로 관리
2. 상어의 현재 위치에서 BFS를 수행해 물고기 찾기(while)
   - 물고기는 가장 가깝고 좌상단에 있는 것 하나만 반환
3. 먹을 물고기 있으면 이동 거리만큼 시간 증가하고 물고기 먹기
4. 상어가 해당 물고기 먹어서 성장 가능하면 크기 증가하고 먹은 물고기 0으로 초기화
5. 더 이상 먹을 물고기 없으면 종료
