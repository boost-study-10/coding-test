### 📖 풀이한 문제

- 백준 14502 - 연구소
- 문제 링크: https://www.acmicpc.net/problem/14502
- 난이도: 골드4
- 걸린 시간: 30분 / 2트

### 🧩 풀이 설계 / 막힌 부분

- 처음 접근 아이디어:
  - 빈 칸 중 3칸을 조합으로 구해서 벽 세우기
  - 벽 세운 연구소에 바이러스 BFS로 퍼뜨리기
  - 안전 구역 구하기
- 고려한 시간복잡도:
  - 벽 세우기: N = 8, M = 8일 때, 64C3 = 41,664
  - 각 경우마다 BFS: 64C3 \* O(N \* M) = 2,659,456
- 막힌 부분 / 해결 방법:
  - n x m인데 n x n으로 처리해서 틀림 ㅎ-ㅎ;

### ⭐️ 문제에서 주로 사용한 알고리즘

- 벽세우기: 백트래킹(조합)
- 바이러스 퍼뜨리기: BFS

### 🧠 대략적인 코드 설명

1. 지도의 각 지점을 순회하면서 빈 칸(0)과 바이러스(2) 위치 저장
2. 모든 빈 칸에 대해 3군데 조합으로 구해 벽 세우기(1)
3. 벽이 3개 세워지면 해당 지도를 기준으로 바이러스 BFS로 퍼뜨리기
   - 모든 바이러스의 위치가 시작점
4. 바이러스가 모두 퍼진 후 안전 구역(0) 개수 세기
5. 안전 영역은 최대값으로 항상 갱신하기
