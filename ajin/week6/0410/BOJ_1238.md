### 📖 풀이한 문제

- 백준 1238 - 파티
- 문제 링크: https://www.acmicpc.net/problem/1238
- 난이도: 골드3
- 걸린 시간: 38분 / 1트

### 🧩 풀이 설계 / 막힌 부분

- 문제: 각 학생이 자신의 마을에서 X번 마을까지 갔다가 다시 돌아올 때, 왕복 시간이 가장 긴 학생의 소요 시간 구하기
- 처음 접근 아이디어:
  - 도로마다 이동 시간이 있는 방향 그래프 문제이고, 간선 가중치가 모두 양수이므로 다익스트라 사용
  - 각 학생에 대해 자신의 마을에서 X번 마을까지 가는 최단 시간과 X번 마을에서 다시 자신의 마을로 돌아오는 최단 시간을 구한 뒤 더해서 왕복 시간 계산하기
- 고려한 시간복잡도:
  - 힙을 사용한 다익스트라는 기본적으로 O((N + M) log N)
  - 각 정점마다 출발 다익스트라와 도착 다익스트라를 총 2N번 수행하게 되면
  - 전체 시간복잡도는 O(N(N + M) log N)
  - 문제 조건이 N <= 1000, M <= 10000이라 가능하지만, 같은 계산이 많이 반복되는 구조임!!!
- 막힌 부분 / 해결 방법:
  - X -> 각 마을 경로는 시작점이 항상 X로 같기 때문에 매번 다시 구할 필요가 없음
  - 각 마을 -> X은 간선을 반대로 뒤집은 그래프에서 X를 시작점으로 다익스트라를 한 번 돌리면 한 번에 구할 수 있음

### ⭐️ 문제에서 주로 사용한 알고리즘

- 다익스트라
  - 가중치 양수 + 최단 거리

### 🧠 대략적인 코드 설명

1. 인접 리스트 형태로 그래프 저장
2. 각 학생 i에 대해 오고 가는 경로 다익스트라로 최단 거리 계산
   - 우선순위 큐는 [현재까지 걸린 시간, 노드 번호] 형태로 관리
   - 큐에서 꺼낸 시간이 이미 기록된 최단 거리보다 크면 무시
   - 더 짧은 경로를 발견했을 때만 거리 갱신 후 큐에 삽입
3. 두 값을 더해 각 학생의 왕복 시간 계산
4. 왕복 시간들 중 최댓값 출력
