### 📖 풀이한 문제

- 백준 2512 - 예산
- 문제 링크: https://www.acmicpc.net/problem/2512
- 난이도: 실버2
- 걸린 시간: 18분 / 1트

### 🧩 풀이 설계 / 막힌 부분

- 문제: 전체 예산 상한이 정해져 있을 때, 각 지방의 요청 예산을 가능한 한 많이 배정한 후 최대 예산 구하기
- 처음 접근 아이디어:
  - 요청 예산 정렬 후
  - 상한액을 하나 정해서 모든 지방에 총합이 전체 예산 이하인지 확인하면서 가능한 최대 상한액 찾기
  - 요청액이 상한 이하이면 그대로 배정하고, 요청액이 상한 초과이면 상한만 배정
- 고려한 시간복잡도:
  - N <= 10,000, 각 요청 예산은 최대 100,000이므로 상한액을 1부터 직접 증가시키며 확인하는 건 불가능
  - 요청 예산 정렬 O(N log N)하고
  - 상한액을 이분탐색하면서 O(log B), 전체 예산 합을 계산하면 O(N)이므로
  - 전체 시간복잡도는 O(N log N + N log B)
    여기서 B는 요청 예산의 최댓값
- 막힌 부분 / 해결 방법: X

### ⭐️ 문제에서 주로 사용한 알고리즘

- 이분탐색

### 🧠 대략적인 코드 설명

1. 요청 예산 배열을 오름차순 정렬
2. 이분탐색 중간값 mid를 현재 상한액으로 설정하고
3. 각 지방 예산에 대해 총 배정 예산 계산
   - 총 배정 예산이 max_budget 이하라면 현재 상한액도 가능하므로 정답 후보 갱신하고 더 큰 상한액도 가능한지 확인
   - 총 배정 예산이 max_budget을 초과하면 현재 상한액이 너무 크므로 작은 상한액 확인
