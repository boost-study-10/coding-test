### 📖 풀이한 문제

- 백준 11779 - 최소비용 구하기 2
- 문제 링크: https://www.acmicpc.net/problem/11779
- 난이도: 골드3
- 걸린 시간: 40분 / 정답 참고

### 🧩 풀이 설계 / 막힌 부분

- 문제: 도시 간 이동 비용이 주어질 때, 시작 도시에서 도착 도시까지의 최소 비용과 이동 경로 구하기
- 처음 접근 아이디어:
  - 최소 비용은 가중치가 모두 양수이므로 다익스트라로 구하기
  - 경로 출력은 모르것다...
- 고려한 시간복잡도:
  - n <= 1,000, m <= 100,000일 때,
  - 우선순위 큐를 사용한 다익스트라는 O((n + m) log n)으로 해결 가능
  - 경로 복원은 도착 도시에서 시작 도시까지 부모를 따라가는 방식이라 O(n)
  - 전체 시간복잡도는 O((n + m) log n)
- 막힌 부분 / 해결 방법:
  - 최단 경로 구하는 방법
    - 최단 거리가 갱신될 때 현재 도시로 오기 직전의 도시를 parent 배열에 저장하고
    - 탐색이 끝난 뒤 도착 도시부터 parent를 따라가며 경로를 역추적하면서 실제 이동 경로를 복원

### ⭐️ 문제에서 주로 사용한 알고리즘

- 최단거리(다익스트라)

### 🧠 대략적인 코드 설명

1. visited 배열을 Infinity로 초기화하고, 시작 도시의 비용을 0으로 설정
2. 우선순위 큐에 [현재까지 비용, 현재 도시] 형태로 시작 도시 삽입
3. 큐에서 비용이 가장 작은 도시를 꺼냄
4. 현재 도시에서 갈 수 있는 다음 도시들을 순회하면서
   - 현재 비용 + 이동 비용이 기존 최단 거리보다 작으면 갱신
   - 우선순위 큐에 새 비용과 함께 삽입
   - parent[next_city] = city로 이전 도시 저장
   - 다익스트라가 끝나면 visited[end]가 최소 비용
5. 도착 도시부터 시작 도시까지 부모를 따라가며 경로 복원
   - 복원한 경로를 뒤집어 시작 도시부터 도착 도시 순서로 출력
