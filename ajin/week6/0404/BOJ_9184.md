### 📖 풀이한 문제

- 백준 9184 - 신나는 함수 실행
- 문제 링크: https://www.acmicpc.net/problem/9184
- 난이도: 실버2
- 걸린 시간: 9분

### 🧩 풀이 설계 / 막힌 부분

- 문제: 주어진 재귀 함수 w(a, b, c)를 여러 입력에 대해 빠르게 계산해 출력하기
- 처음 접근 아이디어:
  - 재귀식은 그냥 구현하고, 계산한 w(a, b, c)의 값을 저장해 중복 계산 줄이기
- 고려한 시간복잡도:
  - a <= 0 || b <= 0 || c <= 0이면 1이고 a > 20 || b > 20 || c > 20이면 w(20, 20, 20)로 고정되므로,
  - 실제로 계산해야 하는 상태는 1 ~ 20 범위의 (a, b, c) 조합으로 제한됨
  - 각 조합은 최대 한 번만 계산되므로 O(20^3)
    따라서 서로 다른 상태 수는 최대 20 _ 20 _ 20 = 8000개
- 막힌 부분 / 해결 방법: X

### ⭐️ 문제에서 주로 사용한 알고리즘

- DP

### 🧠 대략적인 코드 설명

1. 각 입력에 대해 w(a, b, c) 값을 구해 문자열 형태로 정답 배열에 저장
2. w(a, b, c)에서
   1. 하나라도 0 이하이면 1 반환
   2. 하나라도 20 초과이면 w(20, 20, 20) 반환
   3. 현재 (a, b, c) 조합이 이미 계산된 적 있으면 dp에서 바로 반환
   4. a < b && b < c인 경우 계산값 dp에 저장하고 반환
   5. 그 외의 경우도 계산값 dp에 저장하고 반환

- dp는 a, b, c를 문자열 key로 변환해 객체에 저장하는 방식으로 구현
