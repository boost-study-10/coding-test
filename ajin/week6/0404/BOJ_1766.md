### 📖 풀이한 문제

- 백준 1766 - 문제집
- 문제 링크: https://www.acmicpc.net/problem/1766
- 난이도: 골드2
- 걸린 시간: 30분

### 🧩 풀이 설계 / 막힌 부분

- 문제: N개의 문제를 먼저 풀어야 하는 문제와 가능한 한 번호가 작은 문제부터 푸는 순서 구하기
- 처음 접근 아이디어:
  - 먼저 풀어야 하는 문제 관계가 있으므로 위상 정렬 사용
  - 번호가 가장 작은 문제를 가능한 먼저 풀어야 하므로, 진입차수가 0인 문제들을 우선순위 큐에 넣고
  - 항상 가장 작은 번호(낮은 난이도)를 꺼내는 방식으로 구현
- 고려한 시간복잡도:
  - N <= 32,000이고 M <= 100,000 일 때,
  - 우선순위 큐를 사용해 문제를 넣고 각 간선은 한 번씩만 확인하므로 전체 시간복잡도는 O((N + M) log N)
- 막힌 부분 / 해결 방법: X

### ⭐️ 문제에서 주로 사용한 알고리즘

- 위상정렬

### 🧠 대략적인 코드 설명

1. problems에 A번 문제를 풀면 이후 풀 수 있게 되는 문제를 저장하고
   degree에는 B번 문제보다 먼저 풀어야 하는 문제 수를 저장(진입차수)
2. 모든 문제를 확인하면서 진입차수가 0인 문제를 우선순위 큐에 삽입
3. 우선순위 큐에서 가장 번호가 작은 문제를 꺼내 정답에 추가
4. 현재 문제와 연결된 다음 문제들의 진입차수를 1씩 감소하고
   진입차수가 0이 된 문제는 새로 풀 수 있는 문제이므로 우선순위 큐에 삽입
