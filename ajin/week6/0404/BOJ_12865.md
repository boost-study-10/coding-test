### 📖 풀이한 문제

- 백준 12865 - 평범한 배낭
- 문제 링크: https://www.acmicpc.net/problem/12865
- 난이도: 골드5
- 걸린 시간: 19분

### 🧩 풀이 설계 / 막힌 부분

- 문제: 무게와 가치가 있는 물건들이 주어질 때, 최대 무게 K를 넘지 않으면서 배낭에 담을 수 있는 최대 가치 구하기
- 처음 접근 아이디어:
  - 물건을 하나씩 보면서, 현재 배낭 무게 한도에서 해당 물건을 넣는 경우와 넣지 않는 경우를 비교
- 고려한 시간복잡도:
  - N <= 100, K <= 100,000일 때, 각 물건마다 무게 K부터 자신의 무게 W까지 한 번씩 확인하므로 O(K)이고 이를 N개의 물건에 대해 반복하므로 전체 시간복잡도는 O(N \* K)
- 막힌 부분 / 해결 방법: X
  - 오랜만이라 아이디어 떠올리는 데 한세월...

### ⭐️ 문제에서 주로 사용한 알고리즘

- DP(배낭 문제)

### 🧠 대략적인 코드 설명

1. bags[i]를 무게가 i일 때 얻을 수 있는 최대 가치로 저장
2. 각 물건 [W, V]에 대해 반복하면서
   1. 무게를 배낭 최대 무게 K부터 물건 무게인 W까지 감소시키며 탐색
   2. bags[i]와 bags[i - W] + V 중 더 큰 값을 선택해 갱신
      - bags[i]: 현재 물건을 넣지 않는 경우
      - bags[i - W] + V: 현재 물건을 넣는 경우
3. 모든 물건을 반영한 뒤 bags[K] 출력
