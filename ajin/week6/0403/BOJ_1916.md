### 📖 풀이한 문제

- 백준 1916 - 최소비용 구하기
- 문제 링크: https://www.acmicpc.net/problem/1916
- 난이도: 골드5
- 걸린 시간: 14분

### 🧩 풀이 설계 / 막힌 부분

- 문제: N개의 도시와 M개의 버스가 있을 때, 출발 도시에서 도착 도시까지 가는 최소 비용 구하기
- 처음 접근 아이디어: 버스 비용이 모두 양수이므로 다익스트라 사용
- 고려한 시간복잡도:
  - N <= 1,000이고 M < 100,000 이므로 우선순위 큐를 사용하면
  - 다익스트라는 O((N + M) log N)
- 막힌 부분 / 해결 방법: X

### ⭐️ 문제에서 주로 사용한 알고리즘

- 최단거리(다익스트라)

### 🧠 대략적인 코드 설명

1. 각 버스 정보를 인접 리스트 bus에 저장
2. visited 배열을 Infinity로 초기화하고, 출발 도시의 비용을 0으로 설정
3. 우선순위 큐에 [현재까지 비용, 도시 번호] 형태로 출발 도시 삽입
4. 다익스트라 탐색~
   1. 큐에서 가장 비용이 적은 도시를 꺼냄
   2. 현재 도시가 도착 도시라면, 그 비용이 최단 거리이므로 바로 반환
   3. 현재 도시에서 갈 수 있는 다음 도시들을 순회하면서
      - 현재 비용 + 이동 비용이 기존 기록보다 작으면 갱신
      - 갱신된 값을 우선순위 큐에 삽입
5. 탐색이 끝나면 도착 도시까지의 최소 비용 출력
