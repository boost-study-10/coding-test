### 📖 풀이한 문제

- 백준 11404 - 플로이드
- 문제 링크: https://www.acmicpc.net/problem/11404
- 난이도: 골드4
- 걸린 시간: 15분

### 🧩 풀이 설계 / 막힌 부분

- 문제: 모든 도시 쌍(i,j)에 대해, 도시 i에서 도시 j로 가는 최소 비용 구하기
- 처음 접근 아이디어:
  - 모든 정점 쌍 사이의 최단 거리를 구하는 문제이므로 플로이드 워셜 사용
- 고려한 시간복잡도:
  - n <= 100이므로 O(n^3) 가능
- 막힌 부분 / 해결 방법: X

### ⭐️ 문제에서 주로 사용한 알고리즘

- 최단거리(플로이드 워셜)

### 🧠 대략적인 코드 설명

1. bus를 i번 도시에서 j번 도시로 가는 최소 비용이라고 두고, 인접 행렬을 Infinity로 초기화
2. 버스 정보에서 같은 출발점과 도착점에 대해 여러 버스가 있을 수 있으므로 최소 비용만 저장
3. 자기 자신으로 가는 비용 bus[i][i]는 0으로 설정
4. k를 경유지로 두고, 모든 (i, j) 쌍에 대해
   - bus[i][k] + bus[k][j]과 bus[i][j] 중 더 작은 값으로 갱신
5. 이후 Infinity인 값은 도달할 수 없는 경우이므로 0으로 변경하고 출력
