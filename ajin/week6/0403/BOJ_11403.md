### 📖 풀이한 문제

- 백준 11403 - 경로찾기
- 문제 링크: https://www.acmicpc.net/problem/11403
- 난이도: 실버1
- 걸린 시간: 10분

### 🧩 풀이 설계 / 막힌 부분

- 문제: 가중치 없는 방향 그래프가 주어질 때, 모든 정점(i, j)에 대해 i에서 j로 가는 경로가 존재하는지 구하기
- 처음 접근 아이디어:
  - 입력과 출력이 모두 인접 행렬 형태이고, 모든 정점 쌍에 대한 도달 가능 여부를 구해야 하므로
  - 플로이드-워셜 로직으로 i -> k -> j로 갈 수 있다면 i -> j도 갈 수 있다고 업데이트하기
- 고려한 시간복잡도:
  - N <= 100 이므로 O(N^3) 가능
- 막힌 부분 / 해결 방법: X

### ⭐️ 문제에서 주로 사용한 알고리즘

- 최단거리(플로이드 워셜)

### 🧠 대략적인 코드 설명

1. k를 경유지로 두고, 모든 (i, j) 쌍을 순회
2. i -> k -> j 경로가 존재하면 i -> j 역시 갈 수 있으므로 graph[i][j] = 1로 갱신
3. 모든 경유지를 반영한 뒤, 최종 인접 행렬 출력
