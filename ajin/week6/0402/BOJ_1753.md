### 📖 풀이한 문제

- 백준 1753 - 최단경로
- 문제 링크: https://www.acmicpc.net/problem/1753
- 난이도: 골드4
- 걸린 시간: 20분 / 2트

### 🧩 풀이 설계 / 막힌 부분

- 문제: 방향 그래프가 주어질 때, 시작점 K에서 다른 모든 정점으로 최단 경로 구하기
- 처음 접근 아이디어:
  - 가중치가 양수이고 정점 수가 20,000이므로 다익스트 사용
- 고려한 시간복잡도:
  - V <= 20,000이고 E <= 300,000이므로 모든 간선을 완전탐색하는 방식은 불가능
  - 우선순위 큐를 사용한 다익스트라는 O((V + E) log V)로 해결 가능
- 막힌 부분 / 해결 방법: X
- 틀린 이유: 출력 형식 잘못 처리함 ㅎ.ㅎ

### ⭐️ 문제에서 주로 사용한 알고리즘

- 다익스트라

### 🧠 대략적인 코드 설명

1. 인접 리스트 형태로 그래프 저장하고
2. visited 배열을 모두 Infinity로 초기화하고, 시작 정점 K의 거리를 0으로 설정
3. 우선순위 큐에 [비용, 정점] 형태로 시작 정점 삽입
4. 큐에서 현재 비용이 가장 작은 정점을 꺼내고
5. 현재 정점에서 갈 수 있는 다음 정점들을 순회하면서
   - 현재까지의 비용 + 간선 비용이 기존 최단 거리보다 작으면 갱신
   - 갱신된 값을 다시 우선순위 큐에 삽입
6. 탐색이 끝나면 각 정점의 최단 거리를 확인 7.도달할 수 없는 정점은 "INF"로 변환해 출력
