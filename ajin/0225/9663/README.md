### 📖 풀이한 문제

- 백준 9663 - N-Queen
- 문제 링크: https://www.acmicpc.net/problem/9663
- 난이도: 골드4
- 걸린 시간: 20분

### ⭐️ 문제에서 주로 사용한 알고리즘

백트래킹

### 🧩 풀이 설계 / 코드 설명

#### 시간 복잡도

1. 부르트포스
   - N = 14일 때,
     - 각 행마다 퀸을 놓을 열을 선택한다고 생각하면 14^14..
     - 각 행에 1개만 둔다고 처리해도 N!
   - 백트래킹으로 행을 채워가면서 가능한 위치를 열, 대각선으로 조건처리한다면 가능할지도..,

#### 설계

> N×N 체스판에 퀸 N개를 서로 공격하지 않게 두는 경우의 수

- 퀸이 공격하는 방향
  - 같은 행(row)
  - 같은 열(col)
  - \ 대각선(좌상↘우하)
  - / 대각선(우상↙좌하)

1. 재귀 깊이 cnt를 현재 행 r로 사용
   - 한 행에 퀸 1개 배치
2. 각 행 r에서 확인
   - col[c] : 같은 열 사용 여부
   - dir1[r + c] : r+c 대각선 사용 여부
   - dir2[r - c + n - 1] : r-c 대각선 사용 여부
3. 가능하면 퀸을 배치하고 다음 행으로 진행
4. N개의 퀸을 모두 배치한 경우 개수 추가

#### 대각선 인덱싱

- dir1[r + c]: r+c가 같은 대각선( / 방향 )
- dir2[r - c + n - 1]: r-c가 같은 대각선( \ 방향 )
  - r - c는 음수가 될 수 있으니 배열 인덱스를 위해 +(n-1)을 더함
