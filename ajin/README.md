### 아맞다

#### 이차원 배열 복사하기

```js
const new_arr = arr.map((row) => [...row]);
```

#### 이차원 배열 자르기

```js
board.slice(start_m, end_m).map((row) => row.slice(start_n, end_n));
```

#### 배열 요소의 합

```js
const sum = arr.reduce((a, b) => a + b);
```

#### some, every 활용하기

```js
arr.some((value) => 조건); // 하나라도 만족하면 true
arr.every((value) => 조건); // 모두 만족해야 true
```

#### flatMap 활용하기

1. 조건에 따라 값 생성/제거하기

   ```js
   [1, 2, 3, 4].flatMap((x) => (x % 2 === 0 ? [x] : [])); // [2, 4]
   ```

2. 문자열 분해하기

   ```js
   ["hello world", "hi"].flatMap((str) => str.split(" ")); // ["hello", "world", "hi"]
   ```

3. 값 여러 개로 확장하기

   ```js
   [1, 2, 3].flatMap((x) => [x, x * 10]); // [1, 10, 2, 20, 3, 30]
   ```
